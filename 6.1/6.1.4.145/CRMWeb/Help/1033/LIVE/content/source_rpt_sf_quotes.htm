Quotes

Use this report to print quotes.

If you run this report from a list without selecting any records, the report generates a customer-ready quote for each quote record in the system. If you run this report from within a specific record, the report generates a customer-ready quote only for the record you were working with.

The data in this report comes from the Quote form.

If some data is missing from the report, see Troubleshoot problems with data not displaying in a report.

Note

This topic documents the default report as it was shipped. To find out if this report has been customized, contact your report administrator or system administrator.
