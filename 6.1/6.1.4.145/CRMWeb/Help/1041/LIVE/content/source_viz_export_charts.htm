Export a user or a system chart

If you want to make advanced customizations that can't be achieved through the chart designer in Microsoft Dynamics CRM, you must update the chart definition file. To update the chart definition, export the chart, edit the chart definition file, and then import it back into Microsoft Dynamics CRM. For more information, see View Data with Visualizations in the Microsoft Dynamics CRM SDK.

  1. On the main page of the list of records, click or tap the Charts bar.

  2. Click or tap the arrow next to the chart name and select the chart that you want to export.

  3. Click or tap the ellipsis (…), and then click or tap Export Chart.

  4. You can then open or save the chart definition file.

  5. If you want to import a chart back into Microsoft Dynamics CRM, click or tap the More Commands icon , click or tap Import Chart, browse to the file you want and then import it.

    When you import a chart definition file, Microsoft Dynamics CRM checks the definition file for the chart ID and name, and the name of the definition file.

    You can either replace the existing chart or save the chart definition file with a new name.

    If two or more charts exist with the same name, you must save the chart definition file with a new name.
