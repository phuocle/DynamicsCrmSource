Download a report

To edit a default report, you can download the report from Microsoft Dynamics CRM to a computer that is set up with the report development environment, and then make the necessary changes.

  1. Follow the steps for the app you're using.

    If using the CRM web application

    1. On the nav bar, click or tap Microsoft Dynamics CRM. Then select a work area: Sales, Service, or Marketing. The work area name appears on the nav bar.

    2. Click the work area name, and then click or tap Reports.

    3. Select the report, and then click or tap Edit.

    If using CRM for Outlook

    1. In the Navigation Pane, click or tap Workplace, and then under My Work, click Reports.

    2. On the Reports tab, in the Records group, click Edit.

  2. On the Actions menu, click Download Report.

    The RDL file for the report contains tags that specify whether the report is a Fetch-based report or an SQL-based report.

  3. Click Save, and specify the location in which to save the file. If you're creating a new report rather than modifying a report, rename the file.

  4. In the Download Complete dialog box, click Close.
