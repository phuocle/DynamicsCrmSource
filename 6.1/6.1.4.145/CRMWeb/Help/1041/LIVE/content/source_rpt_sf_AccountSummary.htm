Account Summary

Use this report to determine historically what has occurred with an account. This report also displays what is scheduled to occur in the future. The report displays a chronological summary for an account, including sales and service activities, notes, and records.

This report cannot display more than 10 accounts at a time.

The data in this report comes from:

  • Activity form

  • Case form

  • Contact form

  • Contract form

  • Invoice form

  • Note form

  • Opportunity form

  • Order form

  • Quote form

If some data is missing from the report, see Run a report

Tip

To quickly view the Account Summary report for an account, open the account record. On the Account tab, in the Data group, click Run Report, and then click Account Summary.

Note

This topic documents the default report as it was shipped. To find out if this report has been customized, contact your report administrator or system administrator.
