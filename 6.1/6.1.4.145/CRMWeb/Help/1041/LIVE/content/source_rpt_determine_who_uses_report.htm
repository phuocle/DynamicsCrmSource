Determine who can use a report

By default, the report is a personal report and can be used only by the author unless shared.

  1. Make sure you have the System Administrator security role or equivalent permissions in Microsoft Dynamics CRM.

    Check your security roles

    1. Follow the steps in View your user profile..

    2. Don't have the correct permissions? Contact your system administrator.

  2. Follow the steps for the app you're using.

    If using the CRM web application

    1. On the nav bar, click or tap Microsoft Dynamics CRM > Settings. Then click or tap Customization > Customize the System.

    If using CRM for Outlook

    1. In the Navigation Pane, click or tap Setting > Customization > Customizations. Then click or tap Customize the System.

  3. Click Components > Reports, and then click or tap the report that you want to share.

  4. Click Actions.

    1. To change the owner, select Assign, and then select the user or team.

    2. To share the report, select Sharing, add the user or team you want to share the report with, and then give them specific permissions. Click Share.

  5. Click Save or Save and Close.

If you use Microsoft Dynamics CRM for Microsoft Office Outlook with Offline Access, by default all reports that you own will be available when you are offline. More information: Work offline with CRM for Outlook.
