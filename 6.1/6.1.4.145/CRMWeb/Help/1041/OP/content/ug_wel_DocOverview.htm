Skype または Lync で通話するための Microsoft Dynamics CRM のセット アップ

組織が Skype または Lync を使用するよう設定されている場合、電話番号をクリックまたはタップし、Microsoft Dynamics CRM 内から取引先担当者に電話できます。この機能は "クリック トゥー コール" と呼ばれます。

システム管理者がこの機能の設定を構成する領域が少しあります。

コンピューターの Microsoft Dynamics CRM から電話を掛けるユーザーの場合、 Skype をインストール するか、 Lync を設定して利用可能にする必要があります。

サポートされる Skype または Lync のバージョン

Lync と Skype がサポートされるバージョンは以下の通りです。

  • Microsoft Lync 2010 または Microsoft Lync 2013

  • Skype 6.0

組織で Skype または Lync を使用するかどうかを設定

組織がクリック トゥー コールで使用するテレフォニー プロバイダーを構成できます。

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]> [設定] をクリックまたはタップします。

      ナビゲーション バーに、[設定] が表示されます。

    2. [設定] > [管理] > [システム設定]クリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで、必要に応じて組織を展開し、[設定] > [システム] > [管理] > [システム設定] をクリックまたはタップします。

  3. [全般] タブの [テレフォニー プロバイダーの設定] の [クリック トゥー コールのプロバイダーの選択] で、[Skype] または Lync を選択します。

  4. [OK] をクリックまたはタップします。

電話番号に使用する組織の接頭辞の設定

Microsoft Dynamics CRM から Skype または Lync 電話を掛ける場合、電話番号は有効な番号形式 (<国/地域コード><局番><番号>) で入力し、保存する必要があります。ただし、Microsoft Dynamics CRM のデータの入力時、ユーザーが電話番号に必要な国または地域の接頭辞を含めない場合があります。

これを解決するため、国/地域コード接頭辞を有効にすると、システムは呼び出す番号に国/地域コードを付加することができます。この接頭語付加を組織全体に適用することも、ユーザーが個人用オプションで国または地域コードを設定することもできます。ユーザーが組織に対する設定とは異なるコードを指定すると、ユーザーの設定が組織の設定より優先されます。

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]> [設定] をクリックまたはタップします。

      ナビゲーション バーに、[設定] が表示されます。

    2. [設定] > [管理] > [システム設定]クリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで、必要に応じて組織を展開し、[設定] > [システム] > [管理] をクリックまたはタップし、次に [システム設定] をクリックまたはタップします。

  3. [全般] タブの [既定の国/地域コードの設定] セクションで、[国/地域コード接頭辞を有効にする] チェック ボックスをオンにし、[国/地域コード接頭辞] で +01181 (81 は日本の国コードです) などの国/地域コードを指定します。

  4. [OK] をクリックまたはタップします。
