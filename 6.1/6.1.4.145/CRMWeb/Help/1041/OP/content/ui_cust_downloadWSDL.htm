開発者リソースの表示またはダウンロード

このページは、Microsoft Dynamics CRM SDK、開発者ツールにリンクする開発者またはサービス エンドポイントの URL を知る必要のある開発者を対象にしています。

組織独自の名前

探索サービスと対話するときは、組織の一意の名前が必要です。

WindowsAzure AppFabric Issuer 証明書

これは、Microsoft Dynamics CRM 統合用の Windows Azure AppFabric ACS を構成するために必要な公開証明書です。

ディベロッパー センター

MSDN の Microsoft Dynamics CRM Developer Center (Microsoft Dynamics CRM デベロッパー センター)には、開発者の役に立つ貴重な情報が含まれます。ここにある SDK には、ドキュメント、サンプル、ツール、技術記事、拡張キット、ビデオ、ブログ、フォーラム、その他が含まれます。このページの各項目の使用の詳細については、 SDK ドキュメントの記事を参照してください。

探索サービス

これは、SOAP プロトコルを使用して Microsoft Dynamics CRM サーバーで入手できる、組織についての情報が提供される Web サービスです。この情報には、各組織の URL が含まれます。

組織のサービス

これは、SOAP プロトコルを使用して、組織のビジネス データおよびメタデータへのアクセスを提供する Web サービスです。

組織のデータ サービス

RESTful API を使用して、組織のビジネス データへのアクセスを提供するオープン データ (OData プロトコル) Web サービスです。このリンクを使用すると、この API を使用してデータにアクセスする方法が記述されている Conceptual Schema Definition Language (CSDL) ドキュメントが開きます。
