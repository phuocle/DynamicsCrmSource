フォームの Web リソースの追加または編集

ユーザーにとってさらに魅力的で便利になるように、フォーム上の Web リソースを追加または編集できます。フォームに追加および編集できる Web リソースの種類は画像、HTML ファイル、または Silverlight コントロールです。

  1. システム管理者またはシステム カスタマイザーのセキュリティ ロール、または同等のアクセス許可があることを確認してください。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[設定] の順にクリックまたはタップします。

      ナビゲーション バーに [設定] と表示されます。

    2. [設定] > [カスタマイズ] > [システムのカスタマイズ] をクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで、必要に応じて組織を展開し、[設定] > [カスタマイズ] > [カスタマイズ] > [システムのカスタマイズ] をクリックするか、またはタップします。

  3. [コンポーネント] で [エンティティ] を展開し、作業するエンティティを展開します。

  4. [フォーム] をクリックまたはタップし、一覧内で種類が [メイン] のフォームを探し、開くエントリをダブルクリックまたはタップし、フォームを編集します。

  5. 既存の Web リソースを追加するには、挿入したい場所のタブをクリックまたはタップし (たとえば、[全般] または [メモ])、[挿入] タブ上で [Web リソース] をクリックまたはタップします。

    または

    Web リソースを編集するには、編集するフォーム タブと Web リソースを選択し、[ホーム] タブで [プロパティの変更] をクリックまたはタップします。

  6. [Web リソースの追加] ダイアログ ボックスまたは [Web リソースのプロパティ] ダイアログ ボックスの [全般] タブで、必須フィールドに適切な情報を入力します。

    具体的には、以下に注意します。

    1. [Web リソース] ボックスで、使用するイメージ、HTML、または Silverlight Web リソースを選択します。

      メモ

      Web リソースを指定したら、[Web リソースのプロパティ] セクションがタブの最下部に表示されます。表示されるオプションは、追加する Web リソースの種類により異なります。これらのオプションには、カスタム パラメーターの指定、レコードのオブジェクトの種類コードをパラメーターとして渡すこと、リソースを説明する代替テキストの指定が含まれます。画像に代替テキストを指定すると、すべてのユーザーがさらに容易に画像にアクセスできます。

    2. [名前] ボックスに、フィールドの一意の名前を入力します。名前には、英数字とアンダースコアのみを使用できます。

    3. [ラベル] フィールドは、指定した名前の種類に自動的に設定されますが、提案されたラベルがニーズを満たしていることを確認し、それに応じて更新します。

  7. [形式] タブでは、表示されるオプションは挿入された Web リソースの種類により異なります。これらのオプションには、表示列および行の数の指定、境界を表示するかどうかの指定、およびスクロール動作の指定が含まれます。

  8. [依存関係] タブが [Web リソースの追加] または [Web リソースのプロパティ] ダイアログ ボックス内に表示される場合、[使用可能なフィールド] の一覧から、Web リソースで必要なフィールドを選択し、[選択したレコードの追加] ボタン をクリックまたはタップして、選択したフィールドを [依存フィールド] の一覧に移動し、[OK] をクリックまたはタップしてダイアログ ボックスを閉じます。

  9. フォームの編集が終了したら、ホームタブで、[保存して閉じる] をクリックまたはタップして、フォームを閉じます。

  10. メイン フォームの表示とイベントの動作をプレビューするには以下を実行します。

    1. [ホーム] タブで [プレビュー] をクリックまたはタップし、[フォームの作成]、[フォームの更新]、または [読み取り専用] を選択します。

    2. [プレビュー] フォームを閉じるには、ファイル メニューの 閉じる をクリックまたはタップします。

  11. カスタマイズが完了したら、公開します。

    • 現在編集中のコンポーネントのみのカスタマイズを公開するには、ナビゲーション ウィンドウで、取り組んでいるエンティティをクリックまたはタップし、その後、[パブリッシュ] をクリックまたはタップします。

    • 非公開のすべてのコンポーネントのカスタマイズを一度に公開するには、ナビゲーション ウインドウで、[エンティティ] をクリックまたはタップし、その後、[操作] ツール バーで、[すべてのカスタマイズの公開] をクリックまたはタップします。

メモ

ソリューションのインストールやカスタマイズの発行は、通常のシステム操作に影響を与える可能性があります。ソリューションのインポートは、ユーザーへの影響が最小限に留まるようにスケジューリングすることを推奨します。
