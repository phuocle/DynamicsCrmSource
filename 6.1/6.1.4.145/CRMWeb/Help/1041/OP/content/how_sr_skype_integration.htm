Skype または Lync で呼び出しを設定します

組織が Skype または Lync を使用するよう設定されている場合、Microsoft Dynamics CRM内から、電話番号をクリックまたはタップして取引先担当者を呼び出すことができます。この機能は "クリック トゥー コール" と呼ばれます。

コンピューター上の Microsoft Dynamics CRM 内からの電話をかけるには、 Skype のインストールが必要です。または Lync のインストール手順をシステム管理者に確認する必要があります。

タブレット PC 用 CRM 内からの電話をかけるには、Skype アプリケーションをダウンロードする必要があります。アプリケーションがないときに電話番号をタップすると、電話をかける前にアプリケーションをダウンロードできます。

Lync と Skype がサポートされるバージョンは以下の通りです。

  • Microsoft Lync 2010 または Microsoft Lync 2013

  • Skype 6.0
