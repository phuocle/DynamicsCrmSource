ドキュメントの場所レコードの作成または編集

SharePoint のドキュメントの場所は、SharePoint のドキュメント ライブラリまたはフォルダーを指す Microsoft Dynamics CRM 内のレコードです。

Microsoft Dynamics CRM レコードのドキュメントを保存するには、ドキュメント ライブラリまたはフォルダーが作成されている必要があります。Microsoft Dynamics CRM がドキュメント ライブラリおよびフォルダーを自動的に作成できない場合は、これらを SharePoint に手動で作成できます。SharePoint にドキュメント ライブラリおよびフォルダーを作成してから、これらの SharePoint ドキュメント ライブラリおよびフォルダーを指すドキュメントの場所レコードを Microsoft Dynamics CRM に作成する必要があります。

  1. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]> [設定] をクリックまたはタップします。ナビゲーション バーに、[設定] が表示されます。

    2. [設定]をクリックまたはタップして、次に、[ドキュメント管理]をクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで [設定] をクリックまたはタップし、[システム] の [ドキュメント管理] をクリックまたはタップします。

  2. SharePoint[ ドキュメントの場所] をクリックまたはタップします。

  3. [新規] をクリックまたはタップします。

  4. 必要に応じて、以下の情報を指定します。

    • 名前: ドキュメントの場所の名前を入力します。この名前は、エンティティ レコードの場所のリストに表示されます。

    • 所有者: 既定では、この場所レコードの所有者として追加されます。

    • 説明。ドキュメントの場所の説明を入力します。

    • URL の種類: 絶対 URL で場所を作成するか、相対 URL で場所を作成するかを選択します。

      • [絶対 URL] を選択し、[絶対 URL] ボックスで SharePoint 内のフォルダーの場所の完全修飾 URL を指定します。

        - または -

      • [相対 URL] を選択します。既存のサイトまたはドキュメントの場所レコードに相対的なドキュメントの場所を作成するには、[相対 URL] で既存の SharePoint サイトまたはドキュメントの場所レコードを選択します。2 つ目のボックスに SharePoint フォルダーの名前を入力します。

    • 関連。[検索] ボタン をクリックまたはタップします。[レコードの検索] ダイアログ ボックスの 検索 の一覧で、見つけるレコードの種類を選択します。Microsoft Dynamics CRM で場所レコードを作成するレコードを検索して選択し、[OK] をクリックまたはタップします。

  5. [保存して閉じる] をクリックまたはタップします。

メモ

ドキュメントの場所をアクティブ化または非アクティブ化するには、[ドキュメントの場所] ページでドキュメントの場所レコードを選択し、[アクティブ化] または [非アクティブ化] をクリックまたはタップします。
