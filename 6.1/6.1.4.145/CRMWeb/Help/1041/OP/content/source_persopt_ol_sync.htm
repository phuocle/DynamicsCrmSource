Outlook 用 Microsoft Dynamics CRM での同期オプションの設定

Outlook と Outlook 用 CRM の間で同期するレコードの種類を選択し、発生する同期の頻度を指定するために、フィルターを使用できます。フィルターを使用すると、同期されるアイテムの種類を指定できます。

Outlook 用 CRM で提供されるフィルターは、アイテムの所有者ユーザーに対して、次のアイテムを同期します。

  • 定期的な予定

  • FAX

  • タスク

  • サービス活動と予定(ユーザーが関係者になっていて、予定を承諾している場合)

  • 電話

  • レター

  • 取引先担当者

  1. Outlook 用 CRM、[ファイル]メッセージで、[CRM] > オプションをクリックまたはタップします。

  2. [個人用オプションの設定] ダイアログ ボックスの [同期] タブをクリックまたはタップします。

    メモ

    web アプリケーションでは、このタブは使用できません。

  3. 取引先企業など、同期した他のレコードの種類を選択するには、[Microsoft Dynamics CRM 項目を Outlook または Exchange と同期] で、[フィルター] リンクをクリックまたはタップしてから、適切なフィルターを変更または追加します。詳細 : 同期する Microsoft Dynamics CRM レコードや他のアイテムの種類の選択

  4. Outlook の連絡先に含まれている会社名を含める場合は、[[会社] フィールドを取引先企業の親会社の名前で更新します] チェック ボックスをオンにします。

    重要

    会社名フィールドに対するそれ以降の変更は同期されません。

  5. 複数のコンピューターでOutlook 用 CRMを実行していて、このタスクを実行しているコンピューターが最も頻繁にオンラインにする場合は、[このコンピューターを同期クライアントとして設定する] チェック ボックスをオンにして、Outlook と Microsoft Dynamics CRM の間の情報が自動的にこのコンピューターに同期されるように指定します。

  6. Microsoft Dynamics CRM と Outlook間の同期の頻度を制限する場合は、[Outlook フォルダーのすべてでCRM のアイテムを同期] チェック ボックスを選択し、各同期サイクル間の分の数値を入力します。

    重要

    CRM 組織の管理者は、同期間で許容される最小時間を設定し、ユーザーが同期間隔を変更できないようにすることもできます。このため、ユーザーには、この設定の構成が不可能な場合があります。

  7. 変更を保存してダイアログ ボックスを閉じるには、[OK]をクリックまたはタップします。

Microsoft Office Outlook 用 Microsoft Dynamics CRM のプライバシー情報

Microsoft Office Outlook 用 Microsoft Dynamics CRM を使用するには、資格情報 (電子メール アドレスとパスワード) を使用してサインインする必要があります。こうした資格情報は、Microsoft Outlook を開くたびに入力しなくても済むように、ローカルに保存することができます。この情報の保存を選択した場合は、Outlook を開くたびに Outlook 用 CRM が自動的に Microsoft Dynamics CRM に接続されます。
初めて Outlook 用 CRM にサインインして使用した後は、インターネット利用時はお客様のコンピューターと CRM が常に接続された状態になります。コンピューターと CRM との接続は、[オプション] 領域でオフにできます。ただし、このオプションを無効にすると、Outlook 用 CRM のパフォーマンスが低下する可能性があります。
Outlook 用 CRM を使用して、CRM の電子メールを追跡できます。その場合、電子メール スレッドはその閲覧のアクセス許可を持つ組織内のユーザーに表示されます。
Outlook 用 CRM は、お客様が受信するすべての電子メールについて、送信者の電子メール アドレス、受信者の電子メール アドレス、およびメッセージの件名を CRM に送信します。これにより、CRM では特定のメールを CRM サービスによって保存する必要があるかどうかを確認できます。アイテムが追跡対象になると、そのアイテムのコピーが CRM サービスによって保持され、適切なアクセス許可を持つ組織内の他のユーザーに表示されるようになります。アイテムの追跡を解除すると、そのアイテムを自分が所有している場合に限り、アイテムのコピーが CRM サービスから自動的に削除されます。
