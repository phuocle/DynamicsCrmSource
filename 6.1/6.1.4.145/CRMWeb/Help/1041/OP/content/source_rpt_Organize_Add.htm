レポートの追加

Microsoft Dynamics CRMの外部でカスタム レポートを作成した場合は、そのレポートを CRMに簡単に追加することができます。

  1. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM] をクリックまたはタップします。次に、[営業]、[サービス]、または [マーケティング] から作業領域を選択します。作業領域の名前はナビゲーション バーに表示されます。

    2. 作業領域名をクリックまたはタップし、[レポート] をクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで [ワークプレース] をクリックまたはタップし、[自分の仕事] の [レポート] をクリックまたはタップします。

  2. [新規] をクリックまたはタップします。

    別のアプリケーションで作成したファイルの追加

    1. [ソース] セクションの [レポートの種類] ボックスで、[既存のファイル] を選択します。

    2. [ファイルの場所] ボックスに、追加するファイルのパスとファイル名を入力するか、[参照] をクリックまたはタップしてファイルを選択します。

    --または

    Web ページへのリンクの追加

    1. [ソース] セクションの [レポートの種類] ボックスで、[Web ページへのリンク] を選択します。

    2. [Web ページの URL] ボックスに、Web ページの URL を入力します。

  3. [全般] タブでレポートのプロパティを指定します。

    1. [詳細] セクションで、レポートに対して意味のある名前と説明を指定します。

    2. 現在のレポートの親レポートが存在する場合は、親レポート テキスト ボックスにそれが表示されます。

    3. [カテゴリ]。[このフィールドの値を選択または変更します] ボタン ボタンをクリックまたはタップし、このレポートの分類先となるカテゴリを指定します。

    4. 関連するレコードの種類。特定のレコードの種類のページの [レポート] 一覧にレポートを表示するには、[このフィールドの値を選択または変更します] ボタンをクリックまたはタップし、レコードの種類を選択します。

    5. 表示形式。レポートが表示される場所を指定するには、[このフィールドの値を選択または変更します] ボタンをクリックまたはタップし、1 つまたは複数のオプションを選択します。

      値を選択しない場合、レポートはエンドユーザーに対して表示されません。

  4. [管理] タブでレポートの所有者情報を編集します。

    • 所有者

      このオプションは、レポートに対する Assign のアクセス許可がある場合にのみ使用できます。

    • 公開対象

      レポートを組織所有にして組織内の全員に公開するには、[組織] を選択します。所有者、および所有者とレポートを共有するユーザーにのみレポートを公開するには、[個人] を選択します。

  5. [保存] または [保存して閉じる] をクリックまたはタップします。

新しいレポートを共有するには、「 他のユーザーまたはチームとレポートを共有」の説明を参照してください。
