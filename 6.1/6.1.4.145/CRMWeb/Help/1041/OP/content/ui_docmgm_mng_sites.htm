サイト レコードの作成または編集

Microsoft Dynamics CRM 内のサイトは、SharePoint 上のサイトまたはサイト コレクションを指すレコードです。SharePoint 内のサイトおよびサイト コレクションには、Microsoft Dynamics CRM レコードのドキュメントの保存に使用するドキュメント ライブラリおよびフォルダーが含まれます。

サイト レコードの作成

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションの手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[設定] の順にクリックまたはタップします。

      ナビゲーション バーに [設定] と表示されます。

    2. [設定]、[ドキュメント管理] の順にクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウの [設定]  をクリックします。

    2. [システム] の下で [ドキュメント管理] をクリックします。

  3. [SharePoint サイト] > [新規] をクリックします。

  4. 次の情報を入力します。

    • 名前: サイトのわかりやすく使いやすい名前を指定します。

    • 所有者: 既定では、サイト レコードの所有者として追加されます。

    • 説明。サイトの説明を指定します。たとえば、サイトにどのようなドキュメントが格納されるかを指定します。

    • URL の種類: サイトの絶対 (完全) URL を追加するか、相対 URL を追加するかを選択します。

      • 絶対 URL: このサイト レコードが SharePoint 内のサイト コレクションまたはサイトを指すようにするには、サイト コレクションまたはサイトの完全修飾 URL を指定します。このレコードを親サイトとして使用して、同じ SharePoint サイト上のサイト コレクションまたはサイト内にサイトの相対 URL を持つ他のサイト レコードを作成できます。

      • 相対 URL: このオプションは、SharePoint 内のサイト コレクションを指すサイト レコードが少なくとも 1 つある場合にのみ使用してください。[親サイト] ボックスで、既存の Microsoft Dynamics CRM サイト レコードを選択します。親サイトとして選択したサイト レコードが SharePoint 上のサイト コレクションを指す場合は、2 つ目のボックスで既存サイトの名前を指定します。親サイトとして選択したサイト レコードが SharePoint 上のサイトを指す場合は、SharePoint 上の既存の下位サイトの名前を指定します。

    • リスト コンポーネントがインストール済み: SharePoint Server 2010 または SharePoint Server 2013 上のサイト コレクションの URL を指定した場合、およびサイト コレクションに Microsoft Dynamics CRM リスト コンポーネントがインストールされている場合は、このチェック ボックスをオンにします。

      このチェック ボックスは、[絶対 URL] オプションを使用した場合にのみ表示されます。

      [URL の検証] セクションには、URL が最後に検証されたときの検証の状態と日付が表示されます。

  5. [保存]  をクリックします。

  6. サイトの URL を検証するには、[操作] グループの [検証] をクリックします。[保存して閉じる] をクリックします。

  7. [保存して閉じる] をクリックします。

サイト レコードの編集

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションの手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[設定] の順にクリックまたはタップします。

      ナビゲーション バーに [設定] と表示されます。

    2. [設定]、[ドキュメント管理] の順にクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウの [設定]  をクリックします。

    2. [システム] の下で [ドキュメント管理] をクリックします。

  3. サイト レコードを編集するには、[SharePoint サイト] ページでサイト レコードを選択し、[レコード] グループで [編集] をクリックします。

  4. [保存]  をクリックします。

  5. サイトの URL を検証するには、[操作] グループの [検証] をクリックします

  6. [保存して閉じる] をクリックします。

メモ

サイト レコードをアクティブ化または非アクティブ化するには、[SharePoint サイト] ページでサイト レコードを選択し、[レコード] グループで [アクティブ化] または [非アクティブ化] をクリックします。
