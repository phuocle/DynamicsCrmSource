サポート案件の作成または編集

Microsoft Dynamics CRM でサポート案件を使用して、顧客が直面している問題を追跡します。サポート案件を契約および契約品目に関連付けて、顧客に提供することができるサービス レベルを素早く認識し、サポート案件の解決に役立つ記事に情報を見つけたら、サポート情報記事に関連付けます。

サポート案件の作成

  1. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[サービス] の順にクリックまたはタップします。

    2. ナビゲーション バーに [サービス] と表示されます。

    3. 次に、[サービス]、[サポート案件] の順にクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. 自分の組織をクリックまたはタップします。

    2. [サービス] > [サービス] > [サポート案件] をクリックまたはタップします。

  2. コマンド バーで、[新しいサポート案件] をクリックします。

  3. テキスト ボックスに情報を入力します。

    hovertipは、何を入力すべきかについてのヒントを提供します。

    1. 顧客とサポート案件の詳細を見つけるには、プロセス バーで、[識別する] をクリックします。詳細 :Track the customer and case details.

    2. 解決策を持つ他の類似したサポート案件があるかどうかを確認するには、[リサーチ] をクリックします。詳細 :Find a solution from other similar cases

    3. 顧客に問題のソリューションを与えた場合は、サポート案件を解決します。詳細 :Resolve or cancel a case

    4. 顧客との会話を追跡するには、活動を追加します。詳細 : 電話、タスク、電子メール、または予定をサポート案件に追加する

      活動を完了する際には必ず、その活動に要した時間を 期間 ボックスに記録してください。このサポート案件が契約品目にリンクしている場合は、このサポート案件に関するすべての活動期間が計算され、関連するアクティブな契約で自動的に更新されます。その契約に関連するその他のサポート案件の合計を含めた合計を、顧客に請求する前に手動で調整できます。

    5. [契約と製品情報] で、サポート案件に適用される契約および契約品目に関する情報を入力します。契約および契約品目をサポート案件にリンクすることで、サービス単位の使用を追跡し、顧客が適切なレベルのサービスを受けているかどうかを確認できます。該当するすべての製品情報も入力します。

    6. [サポート技術情報の記事] の [サポート情報の記事] で、検索アイコンをクリックし、記事にこのサポート案件を解決する情報が含まれている場合、その記事を選択します。

  4. データを保存する準備ができたら、[保存] をクリックします。

サポート案件の編集

  1. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[サービス] の順にクリックまたはタップします。

    2. ナビゲーション バーに [サービス] と表示されます。

    3. 次に、[サービス]、[サポート案件] の順にクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. 自分の組織をクリックまたはタップします。

    2. [サービス] > [サービス] > [サポート案件] をクリックまたはタップします。

  2. 一覧からサポート案件を開きます。

  3. テキスト ボックス内の情報を変更します。

    hovertipは、何を入力すべきかについてのヒントを提供します。

  4. データを保存する準備ができたら、[保存] をクリックします。
