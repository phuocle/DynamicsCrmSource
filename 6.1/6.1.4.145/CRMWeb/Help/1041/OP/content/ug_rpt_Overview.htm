レポートの既定のフィルターの編集

レポートが Microsoft SQL Server Reporting Services レポートであり、事前のフィルター適用が有効になっていて、既定のフィルターが定義されている場合、レポートで表示が想定されるデータを表示するために、既定のフィルターを変更できます。このフィルターは、ユーザーがレポートを実行するたびに使用されます。

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。.

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM] をクリックまたはタップします。次に、[営業]、[サービス]、または [マーケティング] から作業領域を選択します。作業領域の名前はナビゲーション バーに表示されます。

    2. 作業領域の名前をクリックまたはタップし、[レポート] クリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. 自分の組織をクリックまたはタップします。

    2. 作業領域、[営業]または[サービス]あるいは [マーケティング] をクリックまたはタップし、[ツール] > [レポート] をクリックまたはタップします。

  3. レポートを選択し、コマンド バーで、[そのほかのコマンド]をクリックまたはタップしてから、[既定のフィルターの編集] をクリックまたはタップします。

    既定のビューでは表示されないサブレポートを含め、すべてのレポートを表示するには、すべてのレポート (サブレポートを含む) ビューを選択します。

  4. フィルター条件を変更します。

    フィルター条件は、[取引先企業] や [取引先担当者] など、フィルターで使用できるレコードの種類によってグループ化されます。

    • 既存の行を編集するには、クエリ関係演算子をクリックまたはタップして演算子を選択するか、下線の付いた値をクリックまたはタップして新しい値を入力します。

    • クエリ関係演算子をクリックまたはタップして、演算子を選択します。

    • 条件行を追加するには

      1. [選択] をクリックまたはタップして、フィルター処理するフィールドを指定します。

      2. クエリ関係演算子をクリックまたはタップして、演算子を選択します。

      3. [値の入力] をクリックまたはタップし、フィルター処理する値を入力します。一部の値については、[このフィールドの値を選択または変更します] ボタン をクリックまたはタップして [値の選択] ダイアログ ボックスをオープンし、目的の値を選択できます。

    • 条件をグループ化するには、レコードの種類が同じである 2 つ以上の行を選択する必要があります。たとえば、[営業段階] と [売り上げ見込み] は、どちらもレコードの種類が [営業案件] のフィールド値であるため、これらのフィールドのフィルター条件を指定する 2 つの行はグループ化することができます。ただし、[取引先企業] と [営業案件] のように、レコードの種類が異なるフィールド値を持つ行はグループ化できません。

      1. グループ化する各行について、詳細モードで、その行の [オプション メニュー] ボタン をクリックまたはタップし、次に [行の選択] をクリックまたはタップします。

      2. [フィルター] ツール バーで、[グループ AND] または [グループ OR] を選択します。

      3. 行をグループから削除するには、その行の [オプション メニュー] ボタン をクリックまたはタップし、次に [削除] をクリックまたはタップします。

      4. グループを選択するには、そのグループの [オプション メニュー] ボタン をクリックまたはタップし、次に [グループの選択] をクリックまたはタップします。

      5. グループにフィルター条件句を追加するには、そのグループの [オプション メニュー] ボタン をクリックまたはタップし、[句の追加] をクリックまたはタップして、フィールド、クエリ関係演算子、および値を選択します。

      6. 既に選択されているグループを選択解除するには、そのグループの [オプション メニュー] ボタン をクリックし、[グループの選択解除] をクリックまたはタップします。

      7. グループを選択解除するには、そのグループの [オプション メニュー] ボタン をクリックまたはタップし、[グループ化の解除] をクリックまたはタップします。

      8. [グループ AND] グループを [グループ OR] グループに、または [グループ OR] グループを [グループ AND] グループに変更するには、そのグループの [オプション メニュー] ボタン をクリックし、次に [OR に変更] または [AND に変更] をクリックまたはタップします。

        ヒント

        • すべての条件をクリアしてもう一度やり直すには、[フィルター] ツール バーで [クリア] をクリックまたはタップし、次に [確認] をクリックまたはタップします。

        • 行を削除するには、その行の [オプション メニュー] ボタン をクリックまたはタップし、[削除] をクリックまたはタップします。

  5. [既定のフィルターの保存] をクリックまたはタップします。
