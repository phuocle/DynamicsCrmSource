Outlook 用 Microsoft Dynamics CRM でのアドレス帳オプションの設定

Outlook のアドレス帳に含まれている名前および電子メール アドレスに基づいて、電子メール メッセージを取引先担当者および Microsoft Dynamics CRM の他のレコードの種類にリンクできます。

電子メール メッセージを送信または受信すると、差出人および宛先のフィールドに含まれている名前および電子メール アドレスが、選択した Microsoft Dynamics CRM レコードの種類に含まれている名前および電子メール アドレスと比較されます。一致するものがあれば、電子メール メッセージがレコードに対する活動として組み込まれます。

  1. 使用しているアプリケーションに対応した手順を実行してください。

    Microsoft Outlook 2010 の場合

    1. [ファイル] タブで、[CRM] > [オプション]をクリックまたはタップします。

    Microsoft Outlook 2013 の場合

    1. [ファイル] タブで、[CRM] > [オプション]をクリックまたはタップします。

  2. [個人用オプションの設定] ダイアログ ボックスの [アドレス帳] タブをクリックまたはタップします。

    メモ

    web アプリケーションでは、このタブは使用できません。

  3. [取引先担当者] については、アドレス帳の名前および電子メール アドレスを Microsoft Dynamics CRM の一部の取引先担当者と比較するか、全部の取引先担当者と比較するかを指定します。

    • Microsoft Dynamics CRM に同期された取引先担当者とのみ照合する。既定では、このオプションが選択されます。電子メール メッセージを、アドレス帳の名前または電子メール アドレスと同期すると共に一致させることにした Microsoft Dynamics CRM の取引先担当者レコードにリンクする場合は、このオプションを選択した状態にしておきます。詳細 : 同期する Microsoft Dynamics CRM レコードや他のアイテムの種類の選択

    • Microsoft Dynamics CRM 内のすべての取引先担当者を照合する。電子メール メッセージを、アドレス帳の名前または電子メール アドレスと一致させる Microsoft Dynamics CRM 取引先担当者レコードにリンクするには、このオプションを選択します。

  4. [その他のレコードの種類] については、他のレコードの種類の名前および電子メール アドレスがアドレス帳の名前および電子メール アドレスと一致する場合に、それら他のレコードの種類に電子メール メッセージをリンクするかどうかを選択します。

    • 照合しない。電子メール メッセージを他のレコードの種類にリンクしない場合は、このオプションを選択します。

    • [自分が所有する項目のみ照合する] または [Microsoft Dynamics CRMのすべての項目を照合する]。電子メール メッセージを他のレコードの種類にリンクする場合は、これらのオプションのいずれかを選択します。

  5. [アドレス帳と同期するレコードの種類を変更する] で、[このフィールドの値を選択または変更します] ボタン をクリックまたはタップします。

  6. レコードの種類を追加または削除し、[OK] をクリックまたはタップします。

  7. 変更を保存してダイアログ ボックスを閉じるには、[OK]をクリックまたはタップします。

Microsoft Office Outlook 用 Microsoft Dynamics CRM のプライバシー情報

Microsoft Office Outlook 用 Microsoft Dynamics CRM を使用するには、資格情報 (電子メール アドレスとパスワード) を使用してサインインする必要があります。こうした資格情報は、Microsoft Outlook を開くたびに入力しなくても済むように、ローカルに保存することができます。この情報の保存を選択した場合は、Outlook を開くたびに Outlook 用 CRM が自動的に Microsoft Dynamics CRM に接続されます。
初めて Outlook 用 CRM にサインインして使用した後は、インターネット利用時はお客様のコンピューターと CRM が常に接続された状態になります。コンピューターと CRM との接続は、[オプション] 領域でオフにできます。ただし、このオプションを無効にすると、Outlook 用 CRM のパフォーマンスが低下する可能性があります。
Outlook 用 CRM を使用して、CRM の電子メールを追跡できます。その場合、電子メール スレッドはその閲覧のアクセス許可を持つ組織内のユーザーに表示されます。
Outlook 用 CRM は、お客様が受信するすべての電子メールについて、送信者の電子メール アドレス、受信者の電子メール アドレス、およびメッセージの件名を CRM に送信します。これにより、CRM では特定のメールを CRM サービスによって保存する必要があるかどうかを確認できます。アイテムが追跡対象になると、そのアイテムのコピーが CRM サービスによって保持され、適切なアクセス許可を持つ組織内の他のユーザーに表示されるようになります。アイテムの追跡を解除すると、そのアイテムを自分が所有している場合に限り、アイテムのコピーが CRM サービスから自動的に削除されます。
