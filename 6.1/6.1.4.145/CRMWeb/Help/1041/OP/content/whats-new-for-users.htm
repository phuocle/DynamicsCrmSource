Microsoft Dynamics CRM 2013 へようこそ!

Windows Phone、iPhone、Android 向けの新しい CRM アプリケーション

新しいスマートフォン用アプリケーションを使用すると、電話から、重要な顧客情報にアクセスできます。

Windows マーケットプレース、Apple ストア、または Google Play からアプリケーションをダウンロードするか、CRM 管理者 (自分以外が管理者である場合) に自分の組織に関する説明を確認します。

主要な機能:

  • CRM データがモバイル画面用にすばやく表示および最適化されることを確認します。

  • 取引先担当者、タスク、メモとそのほかの関連データを追加および変更します。

  • アクティビティ フィードを表示して、Bing マップ で住所を確認します。Windows Phone のみ。

  • 接続されていない場合でも、最近表示したレコードに戻ります。Windows Phone のみ。

  • このすべては追加のライセンス料なしです。

詳細 : CRM for phones: Set up and use

Windows 8 または iPad 向けの新しい CRM アプリケーション

タブレット PC 用 Microsoft Dynamics CRM を使用すると、どこにいても接続でき、生産的です。Windows 8 デバイスまたは iPad を使用して、外出中であっても、顧客情報のデータを入手できます。すべての予定に準備が間に合い、詳細事項の記憶が新しいうちにメモ、タスク、取引先担当者、取引先企業、および潜在顧客を更新できます。

Windows マーケットプレースまたは Apple ストア からアプリケーションをダウンロードするか、CRM 管理者 (自分以外が管理者である場合) に自分の組織に関する説明を確認します。

以下と連携:

  • Windows 8 (Surface Pro または Surface RT を含む)

  • iPad (第 4 世代)

  • iPad (第 3 世代)

主要な機能:

  • 使用しやすいダッシュ ボードから、活動、取引先企業、取引先担当者、潜在顧客、および営業案件にアクセス。

  • 何度かタップするだけで顧客データを素早く入力。

  • 主な業績評価指標の進行状況をグラフで視覚的に追跡。

  • 取引先担当者と通信するために Skype を使用。

  • ユーザーにとって最も重要なデータを表示するように、リストの個人用ビューにアクセス。

  • アプリのダッシュボードにタイルをピン留めし、必要な情報を瞬時に入手できる。

  • このすべては追加のライセンス料なしです。

新しいビジネス プロセスは共通のシナリオに対応したベスト プラクティスの順守を促進

同じような組織が共通のシナリオに対応したベスト プラクティスに従えるように助ける複数のビジネス プロセス ソリューションについて、Microsoft Dynamics Marketplace を確認します。これらのソリューションは、ビジネスの方法に適合したビジネス プロセスを作成するための最高の開始点を提供することによって、時間の節約に役立ちます。詳細 :  ダウンロードして入手できるビジネス プロセスのソリューション

また、このシステムには複数のインストール済みのビジネス プロセスが付随しています。ユーザーはこれらのビジネス プロセスをそのまま使用するか、または必要に応じて編集できます。詳細 :  既製のビジネス プロセスの追加

ビジネス プロセスの簡単で読みやすい概要については、 eBook: ビジネス プロセスを確認します。この eBook は、ユーザーがビジネス プロセスにアクセスしてすぐに実行できるように設計されています。

ビジネス プロセスを組織の作業方法に合わせる詳細な手順については、 eBook: ビジネス プロセスの変更を確認します。この eBook は管理者または責任者を対象としています。

InsideView を装備したソーシャル インサイトの発表

InsideView を備えたソーシャル インサイトを使用すると、Microsoft Dynamics CRM の取引先企業情報は、3 つの基本的な情報の種類 (データ、インサイト、および関連性) に更新することによって、スタティックからダイナミックへ移行します。InsideView は、30,000 を超えるソースと数百万の企業およびユーザー プロファイルに独自のトライアンギュレーションおよび検証の技術を適用して、営業担当者がより多くの潜在顧客を見つけ出し、より多くの取引に成功し、取引先企業を維持および拡大することに役立つ、関連する正確な企業および取引先担当者の情報を提供します。Microsoft Dynamics CRM は、追加料金なしで、すべての Microsoft Dynamics CRM オンライン プロフェッショナル ライセンスをユーザーに提供します (米国のみ)。

強力な顧客との対話を生み出します。すべての見込顧客のより多い関連性にアクセスします。ソーシャル インサイトは、CRM の採用を促進する一方で、販売担当者の効率と実働性を向上に役立ちます。その理由は、ソーシャル インサイトは、成功に必要なリアルタイムのインサイトをすべて 1 か所で、すなわち、すべて Microsoft Dynamics CRM 内で販売担当者に提供します。 ソーシャル インサイトについての詳細

新しいデザインのユーザー インターフェイスで簡単に移動

新しいユーザー インターフェイスにより、共通タスクはさらに迅速で簡単になります。

  • ナビゲーション ウィンドウがなくなりました。Web アプリケーションを行き来するには、ページ上部のナビゲーション バーを使用します。ナビゲーション バーには「階層リンク」があり、システムで作業している場所を確認できます。詳細 :  eBook: CRM の入門情報

  • データの入力はより簡単です。ページの上部の nav バーで [作成] コマンドを探します。コマンドをクリックまたはタップし、いくつかのフィールドにデータを入力するだけで、システムに情報を入力できます。詳細 :  簡易作成--新しい取引先担当者の入力(または、他のデータ)--素早く

    営業案件、見積もり、受注に対して製品を素早く追加でき、作業中の画面上で、価格、量、値引き額などの詳細を更新できます。さらに、主要な利害関係者を調べて契約に追加すると、関係者とそのロールを一目で確認できます。詳細 :  営業案件の作成または編集

  • クリックまたはタップして誰かに連絡します。電話番号をクリックまたはタップして、Lync または Skype 経由で顧客と連絡します。電子メール アドレスかをクリックまたはタップして電子メールを送信します。また、Bing マップ 上に住所を表示できます。詳細 :  Skype または Lync で呼び出しを設定します

Microsoft Dynamics CRM はタッチ操作対応のモニターにも、マウスが必要なモニターにも適しています。

新しい配色は、タイルの可読性を向上させる

Microsoft Dynamics CRM 2013 更新プログラム ロールアップ 1 は、営業、サービス、マーケティング、設定、ヘルプ領域のタイルに新しい明るい青色を導入し、タイルの行の背景色を、明るい灰色に変更しました。背景色と前景色間のより大きなコントラストにより、タイルを識別して使用するのが簡単になります。


たとえば、新しい明るい灰色の背景により、取引先企業、取引先担当者、潜在顧客、営業案件、サポート案件など、別のレコードの種類のタイルの行を識別するのが簡単になります。さらに、タイルのグループ名を識別するのが簡単になります。例: [自分の仕事]、[顧客]、または [営業]。


大きなコントラストにより、最近表示したレコードの名前を識別するのが簡単になります。

例を見るには: ナビゲーション バーで、[営業] をクリックまたはタップし、[取引先企業] をクリックまたはタップします。[取引先企業] のタイルの下矢印をクリックまたはタップし、最近表示したレコードの新しい明るい色の背景を確認します。
