イメージ Web リソースの追加または編集

Web リソースは、HTML ファイル、JavaScript、および Silverlight アプリケーションなどの複数のファイル形式の仮想ファイルであり、Microsoft Dynamics CRMデータベースに格納され、一意の URL アドレスを使用して取得できます。

  1. Microsoft Dynamics CRM のシステム管理者セキュリティ ロールまたは同等のアクセス許可を持っている必要があります。

    セキュリティ ロールの確認

    1. 「 ユーザー プロファイルの表示」の手順を実行してください。

    2. 適切なアクセス許可を持っていない場合は、システム管理者に問い合わせてください。

  2. 使用しているアプリケーションに対応した手順を実行してください。

    CRM Web アプリケーションを使用する場合

    1. ナビゲーション バーで、[Microsoft Dynamics CRM]、[設定] の順にクリックまたはタップします。

    2. 次に、[カスタマイズ] > [システムのカスタマイズ] をクリックまたはタップします。

    Outlook 用 CRM を使用する場合

    1. ナビゲーション ウィンドウで、[設定] > [カスタマイズ] > [カスタマイズ] をクリックまたはタップします。次に、[システムのカスタマイズ] をクリックまたはタップします。

  3. 新しい Web リソースを作成するには、[コンポーネント] > [Web リソース] をクリックまたはタップしてから、[操作] ツール バーで、[新規] をクリックまたはタップします。

  4. テキスト ボックスに情報を入力または変更します。

    • 名前 テキスト ボックスに入力された名前に、ファイル拡張子を加えたものがファイル名になります。

    • [表示名] テキスト ボックスに入力された名前が [表示名] に表示されます。

    • [種類] ドロップダウン ボックスで、使用するファイルの種類を選択します。

  5. [参照] をクリックまたはタップし、ローカル コンピュータのファイルを選択して、アップロードします。

    • アップロードされているイメージ web リソースをプレビューするには: [Web リソース]> [プレビュー] をクリックまたはタップします。

    • テキスト web リソースを編集するには: [テキスト エディター] をクリックまたはタップします。

    • テキスト web リソースをプレビューするには: [Web リソース]> [プレビュー] をクリックまたはタップします。

  6. データを保存する準備ができたら [保存して閉じる] をクリックまたはタップします。

  7. カスタマイズを公開します。

    • 編集されたコンポーネントのみの場合: [保存] > [ホーム] の [パブリッシュ] をクリックまたはタップします。

    • 非公開のすべてのコンポーネントのカスタマイズを一度に公開するには すべてのカスタマイズを公開 をクリックまたはタップします。

メモ

ソリューションのインストールやカスタマイズの発行は、通常のシステム操作に影響を与える可能性があります。ソリューションのインポートは、ユーザーへの影響が最小限に留まるようにスケジューリングすることを推奨します。
