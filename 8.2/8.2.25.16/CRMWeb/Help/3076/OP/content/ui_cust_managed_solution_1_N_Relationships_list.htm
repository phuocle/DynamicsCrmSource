新增所有 (或許多) 連絡人至 Microsoft Dynamics CRM

如果您想一次加入 20 多個連絡人，或者，如果要新增所有 Outlook 連絡人至 Microsoft Dynamics CRM，該怎麼辦？有該作用的精靈。

  1. 執行精靈之前，請確認您的 Outlook 連絡人資料盡可能完整且準確。例如：

    • 填寫所有遺失資訊，並確認各人的姓名拼字正確。

    • 確認 Outlook 和 Microsoft Dynamics CRM 中的客戶名稱使用相同的拼法。這會讓您輕鬆將連絡人連結至適當的客戶。

    您可能想要將要追蹤的所有連絡人放入單一資料夾，或將連絡人組織到特定資料夾。

  2. 在 [檔案] 功能表上，選擇 [CRM]。

  3. 選擇 [匯入連絡人] 按鈕，然後從下拉式功能表選擇 [新增連絡人]。

  4. 遵循精靈畫面中的步驟進行。如果此程序中有任何地方卡住，選取所有精靈畫面底端的 [深入瞭解將連絡人新增至 CRM] 連結。
