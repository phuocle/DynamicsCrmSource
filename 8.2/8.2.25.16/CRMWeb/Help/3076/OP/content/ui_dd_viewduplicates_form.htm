為什麼要將我的 Outlook 連絡人新增至 Microsoft Dynamics CRM？

如果您已經使用 Outlook 有一陣子，可能有許多連絡人。若要善用 Microsoft Dynamics CRM 和它提供的所有功能，您必須將 Outlook 連絡人新增至 Microsoft Dynamics CRM。這個方式，您可以在同一個地方追蹤所有相關客戶的活動。新增及連結您的 Outlook 連絡人至 Microsoft Dynamics CRM 客戶之後，您將會在 CRM For Outlook 看到與這些連絡人相關聯的電子郵件、工作和約會記錄。您 (或其他人，如果他們可存取您的活動) 也可以在 Microsoft Dynamics CRM 存取這些連絡人記錄。換句話說，兩種運作方式都適用。

此外，如果您同步智慧型手機或其他行動裝置上的 Outlook，您也可以從該裝置存取 CRM 連絡人。

但是，您的個人連絡人又如何呢？

若您的情況與大部分人相似，可能若您的 Outlook 商務連絡人與部分個人連絡人。CRM For Outlook 讓您輕鬆區隔個人連絡人與商務連絡人。除非您決定追蹤他們，您的個人連絡人絕不會出現在 Microsoft Dynamics CRM。
