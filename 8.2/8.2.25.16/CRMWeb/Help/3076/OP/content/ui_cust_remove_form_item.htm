將電子郵件訊息轉換成潛在客戶

現在您已準備好開始建立自己的潛在客戶。如我們之前討論，潛在客戶來自許多不同來源，但建立它們一個最簡單方式是直接從 Outlook 電子郵件訊息。例如，假設某人傳送電子郵件到網站，詢問有關您業務提供的產品或服務。您可將電子郵件訊息轉換成潛在客戶。

  1. 在您的 [收件匣]，選取電子郵件訊息，然後選取功能區上的 [追蹤]。必須追蹤電子郵件訊息，才能轉換它。

  2. 選擇功能區上的 [轉換成]，然後選擇 [潛在客戶]。如您所見，您也可以將電子郵件訊息轉換成商機或案例，但是現在我們將著重於潛在客戶。

  3. 在 [轉換成潛在客戶] 對話方塊，新增連絡人的名字和姓氏與公司名稱。

  4. 選擇 [轉換]。
    CRM For Outlook 會在 Microsoft Dynamics CRM 建立潛在客戶，並開啟 Microsoft Dynamics CRM 潛在客戶表單的該記錄。

  5. 填寫潛在客戶表單的相關資訊。

提示

尋找另一個更輕鬆的方法，從電子郵件訊息建立潛在客戶？當您從 Microsoft Dynamics CRM 資料庫尚未納入的人員接收電子郵件，如果您追蹤電子郵件訊息，該人員的電子郵件地址會在追蹤窗格中顯示成紅色。按一下電子郵件地址，然後選擇 [連絡人] 或 [潛在客戶] 新增至 Microsoft Dynamics CRM。
