設定是否開啟 Outlook 表單 Microsoft Dynamics CRM 或表單

在本指南中，我們會顯示如何選取 [追蹤] 按鈕或 [設定相關] 按鈕 (連絡人的 [設定上層] 按鈕)，來追蹤電子郵件、約會、連絡人及工作。追蹤非常簡單，可讓您使用可能已熟悉的相同 Outlook 介面。

不過若要建立活動記錄或連絡人記錄，您也可以選擇直接開啟 Microsoft Dynamics CRM 表單。如果您使用 Microsoft Dynamics CRM 表單，您不需要追蹤活動以及連絡人。

若某人已自訂 Microsoft Dynamics CRM，使用 Microsoft Dynamics CRM 表單也可能有幫助。例如，CRM 管理員可以新增一般 Outlook 表單中沒有的欄位。

針對新活動使用 Microsoft Dynamics CRM 表單

  • 在 [設定個人選項] 對話方塊，選擇 [一般] 索引標籤，然後在 [選取要用於新活動的表單] 下選取一個或多個核取方塊。例如，若要在建立新工作時自動開啟 Microsoft Dynamics CRM 表單，選取 [工作] 核取方塊。
