使用儀表板開始一天的生活

您可能想要開啟儀表板，開始一天的生活。儀表板提供簡單的清單與圖表，可協助您和您的團隊根據主要計量 (也稱為關鍵效能指標或 KPI) 了解績效。

例如，如果您在銷售中，並開啟銷售儀表板，就會在準銷售案源中看見開啟的商機狀態，或有多少潛在客戶是由您的行銷廣告活動所產生。

  • 若要開啟儀表板，請在左導覽窗格中，開啟您的區域資料夾。例如，開啟銷售資料夾。開啟我的工作子資料夾，然後開啟儀表板子資料夾。
