建立並追蹤寄出電子郵件活動

如果建立新電子郵件，追蹤電子郵件活動基本上是相同程序。當您編寫電子郵件訊息 (或建立新工作或設定約會)，可以直接從電子郵件訊息或約會的 [收件者] 或 [副本] 欄位搜尋 Microsoft Dynamics CRM 連絡人。

  1. 移至您的 [收件匣]，然後選取功能區上的 [新增電子郵件]。

  2. 在 [收件者] 和 [副本] 方塊中輸入一個或多個電子郵件地址。如果要查詢連絡人，請選取 [收件者] 方塊或 [副本] 方塊。新增至 Microsoft Dynamics CRM 的所有連絡人會出現在通訊錄。

  3. 選擇 [追蹤] 按鈕或 [設定相關] 按鈕，如上述。

提示

您可以設定個人選項判斷在通訊錄包含哪些連絡人。例如，您可以包含您擁有的連絡人或可同步處理 Microsoft Dynamics CRM 資料庫的所有連絡人。
