不同記錄類型的圖標位於導覽的第二個層級

導覽第二個層級會將您帶到系統中儲存的不同記錄類型，例如客戶、連絡人、潛在客戶和商機。這是您會找到執行工作所需大部分資訊的位置。

  • 按一下導覽列上的工作區域名稱，然後按一下所需之記錄類型的圖標。按一下向右箭頭 (位於圖標列最右側) 來捲動，並查看更多圖標以找出所有可用的記錄類型。或者，也可以使用滑鼠滾輪來捲動。
