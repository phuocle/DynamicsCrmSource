編輯記錄中的資料

如果要編輯記錄中的資料，該怎麼辦？若要編輯資料，您需要開啟 Microsoft Dynamics CRM 表單。但您無需離開 CRM For Outlook 這麼做。請按兩下清單中的所有記錄開啟 Microsoft Dynamics CRM 表單。

當 Microsoft Dynamics CRM 表單開啟時，您可以編輯資料或執行您在 Microsoft Dynamics CRM 通常會執行的所有其他動作。

提示

若要同時編輯數個記錄，從清單選取您要的記錄，在功能區選取 [編輯]，然後在 [變更多筆記錄] 視窗編輯所要的欄位。
