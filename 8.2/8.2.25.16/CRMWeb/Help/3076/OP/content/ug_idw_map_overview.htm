無論身在何處都有生產力

您可能已經知道這點，但 Microsoft Dynamics CRM 不僅適用於電腦中的網頁瀏覽器，您也可以在電話、Microsoft Surface 或 iPad 上使用它。

這份《CRM 基本知識》指南涵蓋有關在網頁瀏覽器中使用系統的內容，為的就是盡可能使篇幅保持簡短 (而且也因為行動應用程式的運作方式與在 Web 上幾乎相同)。

從您手機或平板電腦的線上商店下載應用程式 (搜尋 "dynamics crm" 以找到應用程式)。您可能需要洽詢您系統的日常管理人員，確保您具有使用應用程式所需的安全性權限。

提示

您也可以視需要，在 Microsoft Outlook 中完成您的 Microsoft Dynamics CRM 工作。請向您的系統管理員確認詳細資料。
