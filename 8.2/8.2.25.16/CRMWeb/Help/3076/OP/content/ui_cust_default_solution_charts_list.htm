自動追蹤接收電子郵件

如本指南稍早所述，追蹤 CRM For Outlook 電子郵件是手動程序。您選取要追蹤的電子郵件，然後選擇 [追蹤] 按鈕或 [設定相關] 按鈕。手動追蹤提供了可讓您將個人電子郵件與 Microsoft Dynamics CRM 電子郵件分開處理的方式。此外，您可以設定自動追蹤電子郵件訊息的選項。

  1. 開啟 [設定個人選項] 對話方塊中，然後選擇 [電子郵件] 索引標籤。

  2. 在 [選取 Microsoft Dynamics CRM for Outlook 與 Microsoft Dynamics CRM 的電子郵件整合方式] 區段，選取 [檢查 Outlook 中的接收電子郵件，然後判斷是否應將電子郵件連結和儲存為 Microsoft Dynamics CRM 記錄] 選項。

  3. 現在您需要決定是否追蹤所有電子郵件或只追蹤特定類型。若要這麼做，請在 [選取要在 Microsoft Dynamics CRM 中追蹤的電子郵件訊息] 區段的追蹤清單中，選取選項。在大多數情況下，您必須選取 [回覆 CRM 電子郵件的電子郵件訊息] 選項。
