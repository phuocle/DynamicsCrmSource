將 Outlook 連絡人新增至 Microsoft Dynamics CRM

您可以新增活動記錄的相同方式，新增連絡人記錄至 Microsoft Dynamics CRM - 透過追蹤它們。本指南稍早，我們顯示如何將活動記錄連結至特定記錄，例如使用 [設定相關] 按鈕的商機或案例。您可以對連絡人執行同樣的事，但是，您將連絡人連結至 Microsoft Dynamics CRM 客戶。

  1. 在 CRM For Outlook 的導覽窗格中，按一下 [人員]。

  2. 最多選取 20 個連絡人 (按住 Shift 鍵選取多個相鄰的連絡人或 Ctrl 鍵選取不相鄰的連絡人)。

  3. 在功能表上，執行下列其中一項動作：

    • 選取 [追蹤] 按鈕複製連絡人記錄至 Microsoft Dynamics CRM，而不將連絡人連結至現有的客戶記錄。

    • 選取 [設定上層] 按鈕複製連絡人記錄至 Microsoft Dynamics CRM，並同時將連絡人連結至客戶記錄。

  4. 如果選取 [設定上層] 按鈕，在 [查詢記錄] 對話方塊中，選取將連絡人連結至的客戶名稱，然後選取 [新增]。

當您追蹤連絡人時，CRM For Outlook 在人員連絡人卡片下方顯示追蹤窗格。追蹤窗格顯示已追蹤的連絡人，並提供客戶記錄的連結，如果您使用 [設定上層] 按鈕追蹤記錄。

提示

如果您希望填入在 Outlook 表單中無法使用的 Microsoft Dynamics CRM 特定欄位，請選取 [在 CRM 中檢視] 按鈕。
