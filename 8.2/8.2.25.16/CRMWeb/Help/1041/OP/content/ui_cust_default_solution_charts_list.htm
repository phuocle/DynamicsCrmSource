受信電子メールの自動追跡

このガイドの前半で説明したように、Outlook 用 CRM での電子メールの追跡は手動のプロセスです。追跡する電子メールを選択し、次に [追跡] ボタンまたは [関連を設定] ボタンを選択します。手動での追跡には、個人的な電子メールを Microsoft Dynamics CRM の電子メールと別々に保存する方法が用意されています。代替策として、電子メール メッセージの自動追跡オプションを設定できます。

  1. [個人用オプションの設定] ダイアログ ボックスを開き、[電子メール] タブを選択します。

  2. Outlook 用 Microsoft Dynamics CRM の電子メールを Microsoft Dynamics CRM と統合する方法を選択してくださいセクションで、受信電子メールを Outlook で確認し、電子メールを Microsoft Dynamics CRM レコードとしてリンクし保存するかどうか判断するオプションをオンにします。

  3. ここで、すべての電子メール メッセージを追跡するか、または特定の種類のみを追跡するか決定する必要があります。そのためには、Microsoft Dynamics CRM で追跡する電子メール メッセージを選択してくださいセクションの [追跡] リストで、オプションを選択します。多くの場合、[CRM 電子メールに対する返信の電子メール メッセージ] を選択します。
