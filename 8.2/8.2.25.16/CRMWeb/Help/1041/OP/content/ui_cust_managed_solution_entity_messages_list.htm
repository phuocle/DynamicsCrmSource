営業提案または価格見積もりの送信

営業サイクルの中で潜在メンバーを移動するとき、製品とその価格の見積もりを含む、詳細な提案を送信する必要があります。
潜在メンバーを前後に移動させるため、システムが履歴を追跡します。

  1. 営業案件の一覧を開き、適切なレコードをダブルクリックして Microsoft Dynamics CRM フォームを開きます。

  2. 製品品目セクションで、[追加] ボタン を選択し、次に製品を選択します。(製品品目 セクションを表示するには、下にスクロールします)

  3. 見積もりセクションで、[追加] ボタン を選択し、次にフィールドに入力して見積もりを作成します。(見積もりセクションを確認するには、下にスクロールする必要があるかもしれません)

  4. コマンド バーで、[アクティブな見積もり] を選択します。

  5. フォームを閉じて、Outlook 用 CRM に戻ります。
