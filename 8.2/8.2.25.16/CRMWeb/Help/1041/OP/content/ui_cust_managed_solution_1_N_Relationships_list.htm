すべて (または多くの) 連絡先を Microsoft Dynamics CRM に追加する

同時に 20 以上の取引先担当者を追加する場合、または自分の Outlook のすべての取引先担当者を Microsoft Dynamics CRM に追加する場合はどうですか。そのためのウィザードがあります。

  1. ウィザードの実行前に、Outlook の取引先担当者データができる限り完全で正確であることを確認します。たとえば、次のようになります。

    • 不足している情報を入力して、人の名前が正しく入力されていることを確認します。

    • その取引先企業の名前が、Outlook および Microsoft Dynamics CRM 内の名前と同じスペルを使用していることを確認します。これにより、その取引先担当者を適切な取引先企業とリンクすることが容易になります。

    また、すべての追跡する取引先担当者を単一フォルダーに配置、または取引先担当者を特定のフォルダーに組織することができます。

  2. [ファイル] メニューで [CRM] を選択します。

  3. [連絡先のインポート] ボタンを選択し、次に [連絡先の追加] を ドロップダウン メニューから選択します。

  4. ウィザード画面の手順に従います。プロセスのいずれかで行き詰まる場合は、いずれかのウィザード画面の下部にある [CRM への取引先担当者の追加の詳細を確認する] リンクを選択します。
