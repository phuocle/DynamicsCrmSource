一覧に列を追加する

一覧にすべての必要な列が含まれていない場合はどうしますか。たとえば、2 番目の勤務先電話番号フィールドを追加する場合があります。列の追加は単純です。

  1. 一覧の任意の列を右クリックします。

  2. [列の追加] を選択します。

  3. 列の表示ダイアログ ボックスの、使用できる列の下で、追加する列を選択し、次に [追加] を選択します。

  4. 追加するすべての列に対して繰り返します。

メモ

一覧から列を削除するには、列を右クリックし、次に [この列を削除] を選択します。
