ダッシュボードから一日を開始する

ダッシュボードを開くことから一日を始めるかもしれません。ダッシュボードには、ユーザーとチームが主な指標 (主要業績評価指数または KPI とも呼ばれる) を処理する方法を確認するために役立つ、読みやすい一覧とグラフが用意されています。

たとえば、営業中に営業ダッシュボードを開くと、営業パイプラインにオープンした営業案件の状態、またはマーケティング キャンペーンにより何件の潜在顧客が生成されたか確認できます。

  • ダッシュボードを開くには、左側のナビゲーション ウィンドウで、自分の領域のフォルダーを開きます。たとえば、営業フォルダーを開きます。自分の仕事サブフォルダーを開き、次にダッシュボード サブフォルダーを開きます。
