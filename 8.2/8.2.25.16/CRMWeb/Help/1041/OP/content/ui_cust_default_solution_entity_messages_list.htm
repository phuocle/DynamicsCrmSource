潜在顧客に営業のパンフレットを送信する

潜在メンバーはあなたの製品に関心があります。彼らは営業のパンフレットを求める電子メールを送信します。営業のパンフレットが Microsoft Dynamics CRM に保管されている場合、最小限の労力で Outlook 用 CRM から直接それを送信できます。

  1. 電子メールがまだ追跡されていない場合、[追跡] ボタンを選択してそれを追跡します。

  2. リボンで、[営業資料] を選択します。

  3. [その他の営業資料] を選択し、取引をクローズするために役立つ営業情報を参照します。

メモ

別の電子メール メッセージに営業資料を最近添付した場合、一覧から使用できます。
