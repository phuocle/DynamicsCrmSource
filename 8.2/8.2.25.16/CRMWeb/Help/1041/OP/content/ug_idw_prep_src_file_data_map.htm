顧客センター

別のレコードの種類のタイルは、セカンド レベルのナビゲーションです

セカンド レベルのナビゲーションでは、取引先企業、取引先担当者、潜在顧客、営業案件など、システムに保存されているさまざまなレコードの種類が表示されます。これで、業務の遂行に必要なほとんどの情報を検索できます。

  • ナビゲーション バーで、作業領域名をクリックし、目的のレコードの種類のタイルをクリックします。スクロールして右の矢印(タイル行の右端)をクリックし、使用可能なすべての種類のレコードのタイルを確認します。また、マウスホイールでスクロールすることもできます。
