Outlook フォームまたは Microsoft Dynamics CRM フォームのどちらを開くか設定する

このガイドでは、[追跡] ボタンまたは [関連を設定] ボタン (取引先担当者の [上位を設定] ボタン) 選択することにより、電子メール、予定、取引先担当者、およびタスクを追跡する方法を説明しました。追跡は非常に簡単で、これを使用すると、使い慣れた Outlook のインターフェイスと同じ操作を使用できます。

ただし、活動レコードまたは取引先担当者レコードを作成するときに、直接 Microsoft Dynamics CRM フォームを開くオプションもあります。Microsoft Dynamics CRM フォームを使用する場合、活動と取引先担当者を追跡する必要はありません。

Microsoft Dynamics CRM がカスタマイズされている場合、Microsoft Dynamics CRM フォームの使用も便利です。たとえば、通常の Outlook フォームでは使用できないフィールドを CRM 管理者が追加している場合があります。

新しい活動に Microsoft Dynamics CRM フォームを使用する

  • 個人用オプションの設定ダイアログ ボックスで、[全般] タブを選択し、次に新しい活動に使用するフォームを選択してくださいの下で、1 つ以上のチェック ボックスをオンにします。たとえば、新しいタスクを作成するときに自動的に Microsoft Dynamics CRM フォームを開くには、[タスク] チェック ボックスをオンにします。
