Начало работы с помощью панели мониторинга

Может потребоваться начать работу, открыв панель мониторинга. Панель мониторинга предоставляет простые для чтения списки и диаграммы, отображающие ход вашей работы и работы вашей рабочей группы с помощью ключевых показателей (также называемых ключевыми показателями эффективности).

Например, если вы занимаетесь продажами и открываете панель мониторинга "Продажи", для вас будет отображаться статус открытых возможных сделок в воронке продаж или число интересов, созданных в результате маркетинговых кампаний.

  • Чтобы открыть панель мониторинга, на левой панели навигации откройте папку соответствующей области. Например, откройте папку Продажи. Откройте вложенную папку Мои задачи, а затем откройте вложенную папку Панели мониторинга.
