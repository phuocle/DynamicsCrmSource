Добавление столбца в список

Что делать, если в списке не отображаются все необходимые столбцы? Например, может потребоваться добавить второе поле рабочего телефона. Добавить столбец просто.

  1. Щелкните правой кнопкой мыши любой столбец в списке.

  2. Выберите Добавить столбцы.

  3. В диалоговом окне Отображение столбцов в разделе Доступных столбцы выберите столбец, который требуется добавить, а затем нажмите кнопку Добавить.

  4. Повторите шаги, если нужно добавить дополнительные столбцы.

Примечание

Чтобы удалить столбец из списка, щелкните правой кнопкой мыши столбец и нажмите кнопку Удалить этот столбец.
