Задание времени открытия форм Outlook или Microsoft Dynamics CRM

В этом руководстве приводятся сведения об отслеживании сообщений электронной почты, встреч, контактов и задач с помощью кнопки Отслеживать или кнопки Установить значение поля "В отношении" (Установить родительскую запись для контактов). Отслеживание очень просто и позволяет работать со знакомым интерфейсом Microsoft Outlook.

Однако, также можно открыть формы Microsoft Dynamics CRM напрямую, если требуется создать запись действия или контакта. Если используются формы Microsoft Dynamics CRM, нет необходимости отслеживать действия и контакты.

Использование форм Microsoft Dynamics CRM также может быть полезно, если другой пользователь работает с приложением Microsoft Dynamics CRM, настроенным в соответствии с его требованиями. Например, администратор CRM может добавлять поля, недоступные в обычных формах Microsoft Outlook.

Использование форм Microsoft Dynamics CRM для новых действий

  • В диалоговом окне Настройка личных параметров перейдите на вкладку Общие, а затем в разделе Выбрать форму для использования в новых действиях установите один или несколько флажков. Например, если требуется автоматически открывать форму Microsoft Dynamics CRM при создании новой задачи, установите флажок Задача.
