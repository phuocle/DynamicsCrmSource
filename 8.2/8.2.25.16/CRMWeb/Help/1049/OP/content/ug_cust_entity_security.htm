Планирование встречи для обслуживания клиента

Иногда может требоваться запланировать встречу для того, чтобы специалист по обслуживанию посетил офис клиента для устранения проблемы. Это можно сделать, добавив действие встречи в обращение.

При каждом планировании встречи с помощью ленты новостей обращения встреча автоматически включается в календарь обслуживания.

  1. Откройте обращение.

  2. В разделе Действия щелкните Дополнительно (…) и нажмите кнопку Встреча. Заполните поля и нажмите кнопку Сохранить.
