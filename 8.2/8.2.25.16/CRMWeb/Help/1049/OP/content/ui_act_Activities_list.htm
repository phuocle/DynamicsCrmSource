Задание количества записей, отображаемых в списках

По умолчанию при просмотр записей в списке CRM для Outlook отображает 50 записей на страницу. Это значение можно изменить для отображения большего или меньшего числа записей.

  • В диалоговом окне Настройка личных параметров перейдите на вкладку Общее, а затем в разделе Задайте количество записей на каждую страницу списка записей выберите требуемое количество.
