Microsoft Dynamics CRM
Центр клиентов

Новый, но привычный интерфейс

Если вы принадлежите к большинству пользователей, вы уже используете Microsoft Outlook как центр для обмена сообщениями, планирования и управления контактами. Но, возможно, вы не знакомы с приложением Microsoft Dynamics CRM и всеми его функциями. В Microsoft Dynamics CRM для Outlook, бесплатной надстройке для Microsoft Outlook, можно использовать уже знакомые функции Microsoft Outlook, чтобы упростить изучение и работу с Microsoft Dynamics CRM.

После установки CRM для Outlook можно отправлять и получать электронную почту, а также создавать задачи и встречи таким же образом, как в Microsoft Outlook. Но надстройка изменяет несколько важных аспектов интерфейса Microsoft Outlook. Например, CRM для Outlook добавляет раздел CRM на вкладку Домашняя страница для сообщений электронной почты, встреч, контактов и задач. Она также добавляет новую вкладку CRM на ленту.

В левой части экрана под папкой "Входящие" электронной почты отображается новая панель навигации с названием организации и папками, которые можно использовать для навигации по CRM для Outlook:

Эти и другие компоненты интерфейса позволяют получить доступ ко всем основным функциям Microsoft Dynamics CRM в Microsoft Outlook.
