Отслеживание сообщений электронной почты, встреч, контактов и задач Outlook

Большая часть работы в Microsoft Dynamics CRM заключается в создании и записи различных типов действий (отправка электронной почты, создание встреч, создание задач), которые требуются в процессе продаж или обслуживания. В CRM для Outlook можно создавать эти типы действий так же, как в Outlook, но для отслеживания действий в Microsoft Dynamics CRM необходимо их отслеживать. Для отслеживания сообщения электронной почты, встречи, контакта или задачи достаточно нажать кнопку Отслеживать на ленте.

При отслеживании сообщения электронной почты, встречи, контакта или записи задачи копия этой записи создается в Microsoft Dynamics CRM. После отслеживания записи можно по-прежнему получить доступ к этой записи из CRM для Outlook, выбрав Просмотреть в CRM. Вы или любой пользователь с доступом к вашим записям действий также можете получить доступ к ним в Microsoft Dynamics CRM. Можно использовать панель отслеживания Microsoft Dynamics CRM внизу отслеживаемой записи, чтобы просмотреть, что запись отслеживается, и быстро получить доступ к связанным записям.

Отслеживание позволяет с легкостью отделить личные записи Microsoft Outlook от записей Microsoft Dynamics CRM, поскольку вы отслеживаете только те записи, которые требуется синхронизировать с Microsoft Dynamics CRM.
