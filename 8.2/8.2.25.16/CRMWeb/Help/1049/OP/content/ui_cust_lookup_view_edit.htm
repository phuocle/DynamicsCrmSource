Установка личных параметров

Можно настроить личные параметры в CRM для Outlook для персонализации работы. Большинство параметров, которые можно настраивать в CRM для Outlook, связаны с синхронизацией и отслеживанием. В этом разделе рассматриваются некоторые наиболее общие параметры. Но сначала давайте рассмотрим, как настроить личный параметр.

  1. В меню Файл выберите CRM.

  2. Нажмите кнопку Параметры.

    CRM для Outlook отобразит диалоговое окно Настройка личных параметров, включающее вкладки для различных типов параметров.
