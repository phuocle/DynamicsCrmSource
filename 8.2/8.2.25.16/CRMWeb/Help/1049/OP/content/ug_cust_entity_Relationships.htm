Центр клиентов

Отслеживание каждого взаимодействия с клиентом с помощью примечаний и других действий CRM

В Microsoft Dynamics CRM используются действия для отслеживания всего общения с клиентами. Например, можно делать примечания, отправлять сообщения электронной почты, делать звонки, настраивать встречи и назначать себе задачи при разрешении сервисного обращения. Все эти мероприятия считаются типами действий.

Хотя отслеживать взаимодействия с клиентами путем добавления действия для каждого сообщения электронной почты, звонка, задачи или встречи важно для всех пользователей в организации, это особенно важно для рабочей группы по обслуживанию.

Пока продавцы и другие сотрудники работают с клиентом, участники этой группы просматривают ленту новостей и историю действий, предпринятых пользователем для разрешения проблемы клиента.

Действия можно включить в отчеты для отслеживания повседневного хода выполнения работ и контроля за соблюдением соглашения об уровнях обслуживания рабочей группой.

  1. Откройте обращение.

  2. Нажмите кнопку Действия, чтобы добавить задачу или звонок в запись. Щелкните Дополнительно (…), чтобы добавить сообщение электронной почты или встречу. Либо щелкните Примечания и начните вводить текст.
