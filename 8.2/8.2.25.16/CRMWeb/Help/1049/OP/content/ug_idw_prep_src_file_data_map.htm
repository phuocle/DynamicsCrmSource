Плитки различных типов записей на втором уровне навигации

На втором уровне навигации можно перейти к различным типам записей, хранящихся в системе, таким как организации, контакты, интересы и возможные сделки. Здесь хранится большинство сведений, необходимых для выполнения работы.

  • На панели навигации щелкните имя рабочей области и щелкните плитку требуемого типа записи. Щелкните клавишу со стрелкой вправо (с правой стороны ряда плиток), чтобы прокрутить страницу и просмотреть дополнительные плитки для всех доступных типов записей. Или можно прокрутить страницу с помощью колесика мыши.
