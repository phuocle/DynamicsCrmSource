Настройка области чтения для добавления или удаления разделов

При выборе записи в списке в области чтения под списком отобразятся сведения об этой записи (доступные только для чтения). Можно настроить область чтения, добавив или удалив разделы. Например, для записей интересов можно добавить или удалить раздел "Маркетинговая информация". Также можно изменить порядок отображения сведений в области чтения.

  1. Откройте список для определенного типа записей. Например, если требуется изменить сведения, отображаемые для записей интересов, откройте список "Интересы".

  2. На вкладке Представление выберите Настройка области чтения.

  3. В диалоговом окне Персонализация области чтения выполните одно из следующих действий.

    • Если необходимо удалить сведения из области чтения, в разделе Отображаемые разделы выберите раздел, который требуется удалить, и нажмите кнопку Удалить.

    • Если необходимо добавить раздел, в разделе Доступные разделы выберите раздел, который требуется добавить, и нажмите кнопку Добавить.

    • Если требуется изменить порядок разделов, выберите раздел и щелкните стрелку вверх или вниз под списком Отображаемые разделы.
