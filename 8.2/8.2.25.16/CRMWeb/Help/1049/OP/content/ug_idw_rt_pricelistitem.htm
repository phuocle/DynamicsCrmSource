Настройка макета панели мониторинга по умолчанию

После настройки системы системный администратор выбирает макет панели мониторинга по умолчанию, который будет отображаться для всех пользователей при первом входе в систему. Если требуется использовать другую панель мониторинга, можно переопределить настройки системы по умолчанию.

  • Откройте требуемую панель мониторинга и щелкните По умолчанию в верхней части экрана.
