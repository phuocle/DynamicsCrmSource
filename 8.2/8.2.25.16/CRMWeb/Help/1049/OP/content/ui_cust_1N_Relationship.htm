Настройка макета панели мониторинга по умолчанию

После настройки системы системный администратор выбирает макет панели мониторинга по умолчанию для рабочей области. Если требуется просмотреть другую панель мониторинга при переходе в рабочую область, можно переопределить значение по умолчанию.

  • Используйте список панелей мониторинга для выбора панелей мониторинга и выберите По умолчанию на ленте.
