Отслеживание действия входящего сообщения электронной почты

Рассмотрим, как это выглядит на практике. Чаще всего потребуется отслеживать действие электронной почты, связанное с контактом, возможной сделкой, организацией, сервисным обращением или другим типом записи, который используется для отслеживания всей истории общения с клиентом. Например, вы получаете сообщение электронной почты от потенциального клиента. Он хочет узнать о продукте или услуге, предлагаемой компанией.

  1. В папке "Входящие" (та же папка "Входящие", которая используется для всех других сообщений электронной почты Microsoft Outlook) выберите сообщение электронной почты. Не требуется открывать сообщение электронной почты для его отслеживания.

  2. Нажмите кнопку Отслеживать или кнопку Установить значение поля "В отношении" на ленте. Помните, что можно нажать любую из этих кнопок, но обычно рекомендуется нажимать кнопку Установить значение поля "В отношении", чтобы можно было связать действие электронной почты с более конкретной записью.

  3. При нажатии кнопки Установить значение поля "В отношении" нажмите кнопку Дополнительно.

  4. В диалоговом окне Поиск записи, если приложение CRM для Outlook уже выбрало тип записей, достаточно выбрать запись в списке, нажав слева от этой записи. После отметки записи нажмите кнопку Добавить.

    После нажатия кнопки ДобавитьCRM для Outlook добавляет панель отслеживания вниз записи. На панели отслеживания представлены ссылки на запись "В отношении" и другие связанные записи, например контакты. Если контакт отображается красным цветом, это означает, что CRM для Outlook не удалось обнаружить контакт в базе данных CRM. Можно щелкнуть имя контакта для добавления контакта или создания интереса. CRM для Outlook также добавит значок Отслеживать в CRM в сообщение электронной почты, чтобы указать, что оно отслеживается.

Подсказка

Другой способ быстро просмотреть отслеживаемую запись — нажать кнопку Просмотреть в CRM на ленте. Также эту кнопку можно использовать для ввода данных в поля, которые доступны только в формах Microsoft Dynamics CRM.
