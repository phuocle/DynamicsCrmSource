Настройка домашней страницы

Можно задать рабочую область по умолчанию и страницу по умолчанию, которая будет отображаться при первом входе в систему.

  1. На вкладке Общие сведения в раскрывающемся списке Область по умолчанию выберите рабочую область.

  2. В раскрывающемся списке Вкладка по умолчанию выберите требуемую домашнюю страницу. Эта страница будет отображаться при каждом входе в систему и при нажатии кнопки Домой на панели навигации.
