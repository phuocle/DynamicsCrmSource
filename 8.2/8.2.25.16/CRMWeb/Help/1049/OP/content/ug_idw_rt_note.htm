Визуализация данных на различных диаграммах панели мониторинга и для других полей

Можно попробовать использовать различные типы диаграмм и посмотреть, как выглядят данные при использовании различных полей для расчетов, прямо в макете панели мониторинга. (Выполнять настройку не требуется.)

  • На панели мониторинга выберите сегмент для просмотра данных в другом типе диаграммы, например на круговой диаграмме или гистограмме. В раскрывающемся списке при желании можно выбрать другое поле для использования в расчетах.
