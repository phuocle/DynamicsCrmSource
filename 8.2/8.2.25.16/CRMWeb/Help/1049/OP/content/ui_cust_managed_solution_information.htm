Просмотр записей и сведений о записи

При открытии папки, например Организации или Интересы, отображается список записей для этого типа записей. Например, если открыт папку Организации, отобразится список клиентов с адресами, номерами телефонов и контактами. Списки содержат представление данных, которое похоже на электронную таблицу, в котором можно просмотреть множество записей одновременно. Используйте списки для поиска и сортировки записей.

При просмотре записи в списке обычно можно просмотреть некоторые, но не все сведения по этой записи. Например, некоторые столбцы могут выходить за пределы экрана или можно просмотреть только некоторые данные в отдельном столбце. Если требуется просмотреть все сведения о записи, выберите ее. Сведения о записи отобразятся в области чтения в списке. Обратите внимание, что сведения в области чтения доступны только для чтения. Это лишь быстрый способ предварительно просмотреть все сведения об определенной записи.
