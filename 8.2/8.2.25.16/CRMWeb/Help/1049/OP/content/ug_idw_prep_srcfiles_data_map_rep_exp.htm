Просмотр и сортировка записей в списках

При щелчке плитки организации, контакта, интереса, возможной сделки и т. д. отобразится список записей клиентов данного типа записей. Список можно отфильтровать для просмотра только интересующих вас записей, таких как активные организации, или только контактов, на которые вы подписаны.

  • Щелкните стрелку вниз рядом с именем списка, чтобы поменять представление. (Можно просмотреть подмножество записей в зависимости от выбранного представления.) Щелкните имя столбца для сортировки.
