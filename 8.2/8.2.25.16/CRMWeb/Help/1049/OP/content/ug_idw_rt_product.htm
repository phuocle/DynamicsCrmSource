Импорт данных из Microsoft Excel или других источников

Для отслеживания всех данных клиентов в одном месте может потребоваться импортировать контакты, интересы или других типы записей в Microsoft Dynamics CRM из других источников, например из программы электронной почты, электронной таблицы или телефона.

Примечание

В зависимости от объема данных, которые требуется импортировать, и других факторов, импорт данных может потребовать некоторого планирования. Возможно, потребуется обратиться к системному администратору за помощью.

Сначала экспортируйте данные в файл. Поддерживаются следующие типы файлов:

  • Значения, разделенные запятыми (CSV)

  • Текст (TXT)

  • Cжатый файл (ZIP)

  • Электронная таблица Microsoft Office Excel 2003 (XML)

Максимальный допустимый размер ZIP-файлов составляет 32 МБ. Для других типов файлов максимальный допустимый размер файлов составляет 8 МБ.

С помощью мастера импорта данных импортируйте данные.

  1. В рабочей области Параметры щелкните плитку Управление данными. (Может потребоваться щелкнуть стрелку вправо справа от строки плиток, чтобы найти эту плитку.)

  2. Щелкните Импорты и нажмите кнопку Импортировать данные. Следуйте инструкциям на экране.
