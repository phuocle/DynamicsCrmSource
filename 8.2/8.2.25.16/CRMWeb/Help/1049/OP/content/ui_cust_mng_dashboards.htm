Что еще можно делать с автономными фильтрами?

Автономные фильтры в CRM для Outlook — очень мощное и гибкое средство. Можно создать отдельный набор фильтров для различных ситуаций. Например, можно создать один набор фильтров для использования во время командировки в один регион, а другой набор фильтров — для использования в другом регионе. Можно легко активировать и деактивировать фильтры в соответствии с потребностями, нажав кнопки в диалоговом окне Параметры перехода в автономный режим.

Также помните, что можно активировать несколько фильтров одновременно, тем самым обеспечив максимальную гибкость. Каждый фильтр просто добавляет критерии при поиске данных, синхронизируемых с компьютером.

Можно использовать строки условий в фильтрах для внесения простых изменений или создать сложные условия. Выбор за вами. Для получения дополнительных сведений о работе со строками условий выполните поиск автономных фильтров в центр клиентов.
