Добавление контактов Outlook в Microsoft Dynamics CRM

Принцип добавления записей контактов в Microsoft Dynamics CRM аналогичен добавлению записи действия — путем их отслеживания. Ранее в этом руководстве рассказывалось, как связать запись действия с определенной записью, например возможной сделкой или обращением, с помощью кнопки Установить значение поля "В отношении". В случае контактов можно сделать то же самое, но вместо этого контакт связывается с организацией Microsoft Dynamics CRM.

  1. В CRM для Outlook выберите Люди на панели навигации.

  2. Выберите не более 20 контактов. (Удерживайте клавишу SHIFT, чтобы выбрать несколько контактов рядом друг с другом, или клавишу CTRL, чтобы выбрать контакты, которые не расположены рядом друг с другом.)

  3. На ленте выполните одно из следующих действий.

    • Нажмите кнопку Отслеживать, чтобы скопировать запись контакта в Microsoft Dynamics CRM без связи контакта с существующей записью организации.

    • Нажмите кнопку Установить родительскую запись, чтобы скопировать запись контакта в Microsoft Dynamics CRM и одновременно связать ее с записью организации.

  4. При нажатии кнопки Установить родительскую запись в диалоговом поле Поиск записи выберите имя организации, с которой требуется связать контакт, а затем нажмите кнопку Добавить.

При отслеживании контакта CRM для Outlook отобразит панель отслеживания под карточкой контакта пользователя. В области отслеживания показано, что контакт отслеживается, и предоставляется ссылка на запись организации при использовании кнопки Установить родительскую запись для отслеживания записи.

Подсказка

Если необходимо заполнить определенные поля в Microsoft Dynamics CRM, которые недоступны в форме Outlook, нажмите кнопку Просмотреть в CRM.
