Создание и отслеживание действия задачи

Добавление и отслеживание действия задачи очень похоже и добавление и отслеживание действия электронной почты или действия встречи. Существует несколько ключевых различий, о которых следует знать:

  • Microsoft Dynamics CRM имеет отдельные действия для звонков, писем и факсов. CRM для Outlook включает эти типы действий в действие задачи, поэтому, например, если требуется создать действие звонка, сначала необходимо создать действие задачи.

  • Только владелец действия задачи может отслеживать задачу. Если требуется назначить задачу другому пользователю, можно отследить задачу перед ее назначением. Пользователь, которому назначается задача, может отслеживать задачу, если она еще не отслеживается.

Создание и отслеживание действия задачи

  1. Выберите Задачи на панели навигации в нижней части окна CRM для Outlook. Можно нажать кнопку Дополнительно (…) для получения доступа к задачам на панели навигации.

  2. На ленте выберите Новая задача.

  3. Введите значения Тема, дата начала и Срок. Также может понадобиться настроить другие параметры, например Статус, приоритет и Напоминание.

  4. Для отслеживания задачи выполните одно из следующих действий.

    • Если требуется отслеживать стандартную задачу, нажмите кнопку Отслеживать или Установить значение поля "В отношении".

    • Если требуется отслеживать задачу как звонок, письмо или факс, щелкните стрелку раскрывающегося списка на кнопке Отслеживать и выберите требуемый тип задачи.

      Подсказка

      В любой момент можно установить запись "В отношении" после отслеживания определенной задачи. Просто выберите задачу и нажмите кнопку Установить значение поля "В отношении".

  5. Если необходимо назначить запись другому участнику рабочей группы, щелкните Управление задачей, выберите Назначить задачу и введите адрес электронной почты в поле Кому.
