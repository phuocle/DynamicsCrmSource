Использование цвета для классификации записей

Если вы знаете, как использовать цвет для классификации записей в Microsoft Outlook, вы знаете, что делать.

  1. В столбце Категории правой кнопкой мыши установите флажок рядом с записью, которую требуется классифицировать.

  2. Выберите категорию цвета.

Подсказка

Если требуется переименовать категории цвета, чтобы они были более информативными, выберите пункт меню Все категории. Чтобы установить один цвет как цвет для применения при установке флажка, выберите пункт меню Настроить быстрый выбор.
