Отправка предложения продажи или предложения с расценками

При перемещении заинтересованного лица по этапам цикла продаж, скорее всего, потребуется отправить подробное предложение, включающее продукты и расценки.
Поскольку может потребоваться возвращаться к предыдущим действиям с заинтересованным лицом, система отслеживает историю работы с ним.

  1. Откройте список возможных сделок, а затем дважды щелкните соответствующую запись, чтобы открыть форму Microsoft Dynamics CRM.

  2. В разделе Элементы линии продуктов нажмите кнопку Добавить и выберите продукты. (Прокрутите страницу, чтобы найти раздел Элементы линии продуктов.)

  3. В разделе Предложения с расценками нажмите кнопку Добавить и заполните поля для создания предложения с расценками. (Для просмотра раздела Предложения с расценками может понадобиться прокрутить страницу вниз.)

  4. В командной строке щелкните Сделать предложение с расценками активным.

  5. Закройте форму для возврата в CRM для Outlook.
