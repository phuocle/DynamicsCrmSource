Поиск правильной панели мониторинга для роли

CRM для Outlook поставляется с несколькими различными макетами панели мониторинга, с помощью которых можно выделить важные данные и показатели результативности. Лучший способ определить, какой макет подходит именно вам, — просмотреть их все.

  • Чтобы просмотреть различные макеты панели мониторинга, щелкните стрелку раскрывающегося списка рядом с именем панели мониторинга и выберите требуемый макет. Например, если требуется просмотреть список Новые возможности на панель мониторинга, выберите "Панель социального мониторинга продаж".

Подсказка

Можно изменить существующую панель мониторинга, если требуется добавить различные списки или диаграммы.
