Добавление нового действия для клиента

Чаще всего потребуется добавлять действия, связанные с контактом, возможной сделкой, организацией, сервисным обращением или другим типом записи, который используется для отслеживания всей истории общения с клиентом. (Кроме того, можно Добавление действия для себя.)

По мере развития отношений между организацией и клиентом вы и другие участники рабочей группы смогут использовать ленту новостей при работе с клиентом для просмотра истории взаимодействий.

Действия можно включить в отчеты для отслеживания повседневного хода выполнения работ.

  1. Сначала откройте запись. Используйте функцию быстрого поиска на панели навигации, чтобы найти имя клиента, а затем откройте запись.

    Либо на панели навигации щелкните имя рабочей области, щелкните плитку типа записи, найдите запись в списке и откройте ее.

  2. Нажмите кнопку Действия, чтобы добавить задачу или звонок в запись. Щелкните Дополнительно (…), чтобы добавить сообщение электронной почты или встречу. Либо щелкните Примечания и начните вводить текст.
