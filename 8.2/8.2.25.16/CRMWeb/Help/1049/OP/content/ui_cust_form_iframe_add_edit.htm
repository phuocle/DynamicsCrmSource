Просмотр подмножества записей

При работе с большим числом записей может быть полезно отфильтровать список записей для отображения только требуемых записей. CRM для Outlook поставляется с несколькими стандартными фильтрами, которые можно выбрать из для каждого типа записей. Например, можно просмотреть все активные учетные записи или только отслеживаемые учетные записи. Эти стандартные фильтры называются системными представлениями.

При выборе представления из списка CRM для Outlook создает вкладку для этого представления и закрепляет ее над списком. Это упрощает переключение между часто используемыми представлениями. Просто выберите представление для фильтрации записей, затем выберите другое представление для фильтрации по-другому.

Если нажать кнопку Прикрепить, представления будут доступны при следующем открытии CRM для Outlook. Благодаря этому наиболее часто используемые сведения будут всегда под рукой.
