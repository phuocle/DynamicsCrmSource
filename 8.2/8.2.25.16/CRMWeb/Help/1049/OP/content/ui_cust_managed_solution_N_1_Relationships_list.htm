Изменение данных в записи

Что делать, если все же требуется изменить данные в записи? Для изменения данных необходимо открыть форму Microsoft Dynamics CRM. Для этого не обязательно выходить из CRM для Outlook. Просто дважды щелкните любую запись в списке, чтобы открыть форму Microsoft Dynamics CRM.

После открытия формы Microsoft Dynamics CRM можно изменить данные или выполнить другие действия, которые вы обычно выполняете в Microsoft Dynamics CRM.

Подсказка

Если необходимо изменить несколько записей одновременно, выберите нужные записи в списке, щелкните Изменить на ленте и измените требуемые поля в окне Изменить несколько записей.
