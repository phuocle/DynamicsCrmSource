Автоматическое отслеживание входящих сообщений электронной почты

Как мы говорили ранее в этом руководстве, отслеживание электронной почты в CRM для Outlook выполняется вручную. Необходимо выбрать сообщение электронной почты для отслеживания и нажать кнопку Отслеживать или Установить значение поля "В отношении". Отслеживание вручную служит для отделения личной электронной почты от сообщений электронной почты Microsoft Dynamics CRM. Вы также можете отслеживать сообщения электронной почты автоматически.

  1. Откройте диалоговое окно Настройка личных параметров и перейдите на вкладку Электронная почта.

  2. В разделе Выберите метод интеграции электронной почты Microsoft Dynamics CRM для Outlook с Microsoft Dynamics CRM выберите параметр Проверьте входящее сообщение электронной почты в Outlook и определите необходимость связи и сохранения сообщения электронной почты как записи Microsoft Dynamics CRM.

  3. Теперь необходимо решить, что требуется отслеживать: все сообщения электронной почты или только определенные типы. Для этого в разделе Выбор сообщений электронной почты для отслеживания в Microsoft Dynamics CRM в списке Отслеживание выберите параметр. В большинстве случаев потребуется выбрать параметр Сообщения электронной почты в ответ на сообщение электронной почты CRM.
