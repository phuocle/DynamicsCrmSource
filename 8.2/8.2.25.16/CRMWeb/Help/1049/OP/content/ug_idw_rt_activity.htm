Использование быстрого поиска для навигации

Хотите быстро искать и переходить к записям? Используйте поле поиска в верхней части экрана, чтобы выполнять поиск требуемых записей.

  1. Введите условие поиска и щелкните значок Поиск. Используйте символ звездочки (*), если нужно включить подстановочный знак.

  2. В списках соответствующих результатов поиска (сгруппированных по типу записей) щелкните запись, чтобы открыть ее. Если отображается множество совпадений, наведите указатель мыши на один из списков, чтобы отобразить полосу прокрутки для перемещения вверх и вниз по списку для просмотра дополнительных совпадающих записей.

    Если требуется просмотреть совпадения только для определенного типа записи, выберите фильтр из раскрывающегося списка.
