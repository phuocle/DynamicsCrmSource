Добавление нового действия путем отслеживания

Добавление действия в CRM для Outlook отличается от добавления действия в Microsoft Dynamics CRM. Как говорилось ранее в этом руководстве, сообщения электронной почты, встречи и задачи — это типы действий в Microsoft Dynamics CRM. При создании одного их этих действий в CRM для Outlook требуется способ регистрации записи действия для ее связи с соответствующими записями в Microsoft Dynamics CRM. Например, может потребоваться связать действие звонка с определенной возможной сделкой по продаже. Для этого используется отслеживание.

При отслеживании сообщений электронной почты, встреч, контактов или записей задач в CRM для Outlook копии этих записей создаются в Microsoft Dynamics CRM и связываются (синхронизируются) с записями Microsoft Outlook. После отслеживания записи можно по-прежнему получить доступ к этой записи в CRM для Outlook, и любой пользователь с доступом к вашим действиям может просматривать эту запись в Microsoft Dynamics CRM.

Для отслеживания сообщения электронной почты, встречи или действия задачи в CRM для Outlook при создании действия нажмите кнопку Отслеживать или Установить значение поля "В отношении" на ленте.

При нажатии кнопки ОтслеживатьCRM для Outlook связывает запись с любыми контактами Microsoft Dynamics CRM, указанными в строках Кому, Копия или Скрытая копия, выполнив поиск записей, содержащих соответствующие адреса электронной почты. Это здорово, но может быть недостаточно для работы. Вы можете работать с теми же контактами в течение долгого периода времени, поэтому связи записи с контактом может быть недостаточно. Для отслеживания записи и ее связи с более конкретной записью, например возможной сделкой или обращением, нажмите кнопку Установить значение поля "В отношении".
