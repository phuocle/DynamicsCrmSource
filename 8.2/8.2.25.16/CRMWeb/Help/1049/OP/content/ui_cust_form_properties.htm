Все вращается вокруг действий

В CRM для Outlook используются действия для отслеживания всего общения с клиентами. Например, можно отправлять сообщения электронной почты, делать звонки, настраивать встречи, назначать задачи или делать примечания при продаже. Все эти мероприятия считаются типами действий.

Чтобы максимально эффективно использовать CRM для Outlook, важно, чтобы все пользователи в организации отслеживали взаимодействия с клиентами путем добавления действия для каждого сообщения электронной почты, звонка, задачи или встречи. Таким образом история клиента будет полной. По мере развития отношений между организацией и клиентом вы и другие участники рабочей группы смогут использовать список действий при работе с клиентом для просмотра полной истории взаимодействий.

Совместное использование информации также позволяет предоставлять интересующие ключевые точки по всей организации. Действия можно включить на панели мониторинга и в отчеты для отслеживания повседневного хода выполнения работ и контроля за работой рабочей группы.
