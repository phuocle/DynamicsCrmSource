ศูนย์บริการลูกค้า

ไทล์สำหรับเรกคอร์ดชนิดต่างๆ จะอยู่ในระดับที่สองของการนำทาง

ระดับที่สองของการนำทางจะนำคุณไปยังเรกคอร์ดชนิดต่างๆ ในระบบ เช่นลูกค้าองค์กร ผู้ติดต่อ ลูกค้าเป้าหมาย และโอกาสทางการขาย ที่นี่ คุณจะพบข้อมูลส่วนใหญ่ที่คุณต้องใช้ในการทำงานของคุณ

  • บนแถบนำทาง ให้คลิกชื่อพื้นที่ทำงาน แล้วจึงคลิกไทล์สำหรับชนิดของเรกคอร์ดที่คุณต้องการ คลิกที่ลูกศรด้านขวา (ที่ด้านบนขวาของแถวไทล์) เมื่อต้องการเลื่อน และดูไทล์เพิ่มเติมสำหรับชนิดของเรกคอร์ดที่พร้อมใช้งานทั้งหมด หรือคุณสามารถเลื่อนด้วยล้อเลื่อนของเมาส์

ถัดไป: ดูและเรียงลำดับเรกคอร์ดในรายการ
