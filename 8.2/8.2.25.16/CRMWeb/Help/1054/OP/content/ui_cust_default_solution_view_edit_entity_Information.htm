เริ่มต้นวันของคุณด้วยแดชบอร์ด

คุณอาจต้องการเริ่มต้นวันของคุณ ด้วยการเปิดแดชบอร์ด แดชบอร์ดให้ง่ายต่อการอ่านรายการ และแผนภูมิซึ่งช่วยให้คุณเห็นว่าคุณและทีมของคุณกำลังทำกับเมตริกหลัก (หรือที่เรียกอีกอย่างหนึ่งว่าตัวบ่งชี้ประสิทธิภาพหลักหรือ Kpi)

ตัวอย่างเช่น หากคุณกำลังขาย และเปิดแดชบอร์ดการขาย คุณจะเห็นสถานะของโอกาสทางการขายในการขายแบบเปิดในไปป์ไลน์ หรือจำนวนลูกค้าเป้าหมายที่เกิดขึ้นจากแคมเปญทางการตลาดของคุณ

  • เมื่อต้องการเปิดแดชบอร์ด ในบานหน้าต่างนำทางซ้าย เปิดโฟลเดอร์สำหรับพื้นที่ของคุณ ตัวอย่างเช่น เปิดโฟลเดอร์ ขาย เปิดโฟลเดอร์ย่อย งานของฉัน แล้วเปิดโฟลเดอร์ย่อย แดชบอร์ด

ถัดไป: ค้นหาแดชบอร์ดที่เหมาะสมสำหรับบทบาทของคุณ
