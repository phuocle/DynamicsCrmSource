แก้ไขข้อมูลในระเบียน

ดังนั้นถ้าคุณ ทำ ต้องการแก้ไขข้อมูลในระเบียน เมื่อต้องการแก้ไขข้อมูล คุณต้องเปิดแบบฟอร์ม Microsoft Dynamics CRM คุณไม่จำเป็นต้องปล่อย CRM for Outlook เมื่อต้องการทำเช่นนั้น เพียงแค่คลิกสองครั้งที่เรกคอร์ดใดๆ ในรายการเพื่อเปิดแบบฟอร์ม Microsoft Dynamics CRM

เมื่อแบบฟอร์ม Microsoft Dynamics CRM เปิดอยู่ คุณสามารถแก้ไขข้อมูล หรือทำการดำเนินการใดๆ ที่คุณทำตามปกติใน Microsoft Dynamics CRM

คำแนะนำ

ถ้าคุณต้องการแก้ไขหลายระเบียนในเวลาเดียว เลือกระเบียนที่คุณต้องการจากรายการ เลือก แก้ไข จาก ribbon และแก้ไขเขตข้อมูลที่คุณต้องการในหน้าต่าง เปลี่ยนแปลงระเบียนหลายระเบียน
