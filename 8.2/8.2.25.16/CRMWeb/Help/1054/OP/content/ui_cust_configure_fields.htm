เพิ่มผู้ติดต่อ Outlook ไปยัง Microsoft Dynamics CRM

คุณสามารถเพิ่มระเบียนที่ติดต่อของคุณไปยัง Microsoft Dynamics CRM ในแบบเดียวกับที่คุณเพิ่มเรกคอร์ดของกิจกรรม — โดยการติดตาม ก่อนหน้านี้ในคู่มือนี้ ดังที่ได้แสดงวิธีการเชื่อมโยงเรกคอร์ดของกิจกรรมกับระเบียนที่เฉพาะเจาะจง เช่นโอกาสทางการขายหรือกรณีและปัญหา โดยใช้ปุ่ม ตั้งค่าเกี่ยวกับ ด้วยผู้ติดต่อ คุณสามารถทำสิ่งเดียวกัน แต่คุณเชื่อมโยงผู้ติดต่อกับบัญชี Microsoft Dynamics CRM แทน

  1. ใน CRM for Outlook เลือก บุคคล ในบานหน้าต่างนำทาง

  2. เลือกผู้ติดต่อได้ถึง 20 (กดค้างไว้ที่แป้น Shift เพื่อเลือกผู้ติดต่อหลายรายที่อยู่ติดกันหรือ Ctrl ในการเลือกผู้ติดต่อที่ไม่ได้อยู่ติดกัน)

  3. บน ribbon ให้เลือกทำอย่างใดอย่างหนึ่งต่อไปนี้

    • เลือกปุ่ม ติดตาม เพื่อคัดลอกเรกคอร์ดผู้ติดต่อไปยัง Microsoft Dynamics CRM โดยไม่มีการเชื่อมโยงผู้ติดต่อกับระเบียนลูกค้าองค์กรที่มีอยู่

    • เลือกปุ่ม ตั้งค่าหลัก เพื่อคัดลอกเรกคอร์ดผู้ติดต่อไปยัง Microsoft Dynamics CRM และเชื่อมโยงระเบียนลูกค้าองค์กรพร้อมกัน

  4. ถ้าคุณเลือกปุ่ม ตั้งหลัก ในกล่องโต้ตอบ มองหาค่าเรกคอร์ด เลือกชื่อลูกค้าองค์กรเพื่อเชื่อมโยงกับผู้ติดต่อ และจากนั้นเลือก เพิ่ม

เมื่อคุณติดตามผู้ติดต่อ CRM for Outlook แสดงบานหน้าต่างการติดตามที่ด้านล่างของบัตรผู้ติดต่อของบุคคลนั้น บานหน้าต่างการติดตามแสดงว่าผู้ติดต่อมีการติดตามและให้การเชื่อมโยงกับเรกคอร์ดลูกค้าองค์กรถ้าคุณใช้ปุ่ม ตั้งค่าหลัก เพื่อติดตามเรกคอร์ด

คำแนะนำ

ถ้าคุณต้องการกรอกข้อมูลในฟิลด์ที่ระบุใน Microsoft Dynamics CRM ที่ไม่พร้อมใช้งานในแบบฟอร์ม Outlook เลือกปุ่ม มุมมองใน CRM
