ติดตามข้อความอีเมลขาเข้าโดยอัตโนมัติ

ในขณะที่เรากล่าวถึงกันก่อนหน้านี้ในคู่มือนี้ การติดตามอีเมลใน CRM for Outlook เป็นกระบวนการด้วยตนเอง คุณได้เลือกอีเมลที่คุณต้องการติดตาม และจากนั้นเลือกปุ่ม ติดตาม หรือ ตั้งค่าเกี่ยวกับ การติดตามด้วยตนเองช่วยให้คุณสามารถเก็บอีเมลส่วนตัวของคุณแยกต่างหากจากอีเมล Microsoft Dynamics CRM ของคุณ อีกวิธีหนึ่งคือคุณสามารถตั้งค่าตัวเลือกการติดตามข้อความอีเมลโดยอัตโนมัติ

  1. เปิดกล่องโต้ตอบ ตั้งค่าตัวเลือกส่วนบุคคล และเลือกแท็บ อีเมล

  2. ภายใต้ เลือกวิธี Microsoft Dynamics CRM สำหรับ Outlook ควรรวมอีเมลด้วยส่วน Microsoft Dynamics CRM เลือก ตรวจสอบอีเมลขาเข้าใน Outlook และตรวจสอบว่าอีเมลจะมีการเชื่อมโยงและบันทึกเป็นตัวเลือก Microsoft Dynamics CRM ระเบียน

  3. ขณะนี้คุณจำเป็นต้องตัดสินใจว่าจะติดตามข้อความอีเมลทั้งหมดหรือเฉพาะบางชนิด เมื่อต้องการทำเช่นนั้น ใน เลือกข้อความอีเมลที่จะติดตามใน Microsoft Dynamics CRM จากนั้น ในรายการ ติดตาม เลือกตัวเลือก ในกรณีส่วนใหญ่ คุณจะต้องเลือกตัวเลือก ข้อความอีเมลในการตอบกลับอีเมล CRM

ถัดไป: สร้างผู้ติดต่อหรือลูกค้าเป้าหมายจากที่อยู่อีเมลที่ไม่รู้จักโดยอัตโนมัติ
