ตั้งค่าว่าจะเปิด Outlook หรือแบบฟอร์ม Microsoft Dynamics CRM

ตลอดทั้งคำแนะนำนี้ เราได้แสดงวิธีที่คุณสามารถติดตามอีเมล การนัดหมาย ผู้ติดต่อ และงาน โดยเลือกปุ่ม ติดตาม หรือปุ่ม ตั้งค่าเกี่ยวกับ (ตั้งค่าหลัก สำหรับผู้ติดต่อ) การติดตามได้อย่างสะดวก และช่วยให้คุณทำงานเดียวกับอินเทอร์เฟซ Outlook ที่อยู่อาจจะคุ้นเคยกับคุณ

คุณยังมีตัวเลือกในการเปิดแบบฟอร์ม Microsoft Dynamics CRM โดยตรง อย่างไรก็ตามเมื่อคุณต้องการสร้างระเบียนกิจกรรมหรือเรกคอร์ดผู้ติดต่อ ถ้าคุณใช้แบบฟอร์ม Microsoft Dynamics CRM ที่คุณไม่จำเป็นต้องติดตามกิจกรรมและผู้ติดต่อของคุณ

การใช้แบบฟอร์ม Microsoft Dynamics CRM ยังมีประโยชน์ถ้าผู้อื่นมีการกำหนด Microsoft Dynamics CRM ตัวอย่างเช่น ตัวผู้ดูแลระบบ CRM อาจได้เพิ่มเขตข้อมูลที่ไม่พร้อมใช้งานผ่านแบบฟอร์ม Outlook ปกติ

ใช้แบบฟอร์ม Microsoft Dynamics CRM สำหรับกิจกรรมใหม่

  • ในกล่องโต้ตอบ ตั้งค่าตัวเลือกส่วนบุคคล เลือกแท็บ ทั่วไป และจากนั้นภายใต้ เลือกฟอร์มที่จะใช้สำหรับกิจกรรมใหม่ เลือกกล่องกาเครื่องหมายอย่างน้อยหนึ่งครั้ง ตัวอย่างเช่น ถ้าคุณต้องการจะเปิดแบบฟอร์ม Microsoft Dynamics CRM โดยอัตโนมัติเมื่อคุณสร้างงานใหม่ เลือกกล่องกาเครื่องหมาย งาน

ถัดไป: กำหนดบานหน้าต่างการอ่านเพื่อเพิ่มหรือเอาส่วนออก
