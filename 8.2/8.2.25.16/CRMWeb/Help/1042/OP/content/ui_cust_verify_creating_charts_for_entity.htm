새 활동을 추적하여 추가

Outlook용 CRM에서 활동을 추가하는 것은 Microsoft Dynamics CRM에서 활동을 추가하는 것과 다릅니다. 이 가이드 앞부분에서 설명했듯이 전자 메일, 약속 및 작업이 Microsoft Dynamics CRM에서 활동 유형입니다. Outlook용 CRM에서 이러한 활동 중 하나를 만들 때 Microsoft Dynamics CRM에서 해당 레코드와 관련되도록 해당 활동 레코드를 기록하는 방법이 필요합니다. 예를 들어, 전화 통화 활동을 특정 영업 기회와 연결할 수 있습니다. 이 연결을 만드는 방법은 추적을 통하는 것입니다.

Outlook용 CRM에서 전자 메일 메시지, 약속 또는 작업을 추적하면 해당 레코드의 복사본이 Microsoft Dynamics CRM에 만들어지고 Outlook 레코드와 연결(동기화)됩니다. 레코드를 추적한 후 Outlook용 CRM에 있는 해당 레코드에 계속 액세스할 수 있으며 사용자 활동에 대한 액세스 권한이 있는 사람은 Microsoft Dynamics CRM에서 해당 레코드를 볼 수 있습니다.

Outlook용 CRM에서 전자 메일, 약속 또는 작업 활동을 추적하려면 활동을 만들 때 리본 메뉴에서 추적 단추 또는 관련 항목 설정 단추를 선택합니다.

추적 단추를 선택하는 경우 Outlook용 CRM는 전자 메일 주소가 일치하는 레코드를 검색하여 레코드를 받는 사람, 참조 또는 숨은 참조 줄에 포함된 Microsoft Dynamics CRM 연락처와 연결합니다. 이렇게 하는 것이 좋지만 충분하지 않을 수 있습니다. 오랜 시간 동안 동일한 연락처와 작업할 수 있으므로 단지 레코드를 연락처와 연결하는 것으로는 충분하지 않을 수 있습니다. 레코드를 추적하고 영업 기회 또는 서비스 케이스 같은 특정 레코드와 연결하려면 관련 항목 설정 단추를 사용합니다.
