열을 목록에 추가

목록에 원하는 모든 열이 포함되지 않는 경우는 어떻게 합니까? 예를 들어, 두 번째 근무처 전화 필드를 추가하고 싶을 수도 있습니다. 열을 추가하는 방법은 간단합니다.

  1. 목록의 열을 마우스 오른쪽 단추로 클릭합니다.

  2. 열 추가를 선택합니다.

  3. 열 표시 대화 상자의 사용할 수 있는 열에서 추가하려는 열을 선택한 다음 추가를 선택합니다.

  4. 추가하려는 열에 대해 반복합니다.

참고

목록에서 열을 제거하려면 열을 마우스 오른쪽 단추로 클릭한 후 이 열 제거를 선택합니다.
