빠른 검색을 사용하여 탐색

레코드를 찾고 탐색하는 빠른 방법을 원하십니까? 화면 맨 위에 있는 검색 상자를 사용하여 관심이 있는 레코드를 검색합니다.

  1. 검색 조건을 입력한 다음, 검색 아이콘 을 클릭합니다. 와일드 카드 문자를 포함하려면 별표(*)를 사용합니다.

  2. 일치하는 검색 결과 목록에서(레코드 종류별로 그룹화) 레코드를 클릭하여 엽니다. 일치하는 목록이 많은 경우 목록 중 하나를 가리키고 스크롤 막대를 위와 아래로 움직여 더 많이 일치하는 레코드를 확인합니다.

    특정 종류의 레코드와 일치하는 항목만 참조하려는 경우 드롭다운 목록에서 필터를 선택합니다.
