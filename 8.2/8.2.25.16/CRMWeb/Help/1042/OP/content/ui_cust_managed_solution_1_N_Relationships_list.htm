모든(또는 많은) 연락처를 Microsoft Dynamics CRM에 추가

한 번에 20개 이상의 연락처를 추가하려는 경우 또는 Outlook 연락처를 Microsoft Dynamics CRM에 추가하려는 경우 어떻게 하십니까? 이를 위한 마법사가 있습니다.

  1. 마법사를 실행하기 전에 Outlook 연락처 데이터가 가능한 한 정확하고 완전한지 확인하십시오. 예를 들면 다음과 같습니다.

    • 누락된 정보를 모두 입력하고 사람들의 이름을 올바르게 입력했는지 확인하십시오.

    • 거래처 이름이 Outlook 및 Microsoft Dynamics CRM에서 철자가 동일한지 확인하십시오. 그러면 연락처를 적절한 거래처에 연결하는 것이 쉬워집니다.

    또한 단일 폴더에 추적하려는 모든 연락처를 저장하거나 연락처를 특정 폴더에 구성할 수도 있습니다.

  2. 파일 메뉴에서 CRM을 선택합니다.

  3. 연락처 가져오기 단추를 선택한 다음 드롭다운 메뉴에서 연락처 추가를 선택합니다.

  4. 마법사 화면의 단계를에 따라 수행합니다. 프로세스 중간에 막히는 곳이 있으면 마법사 화면 맨 아래에서 CRM에 연락처를 추가하는 방법에 대해 자세히 알아보기 링크를 선택합니다.
