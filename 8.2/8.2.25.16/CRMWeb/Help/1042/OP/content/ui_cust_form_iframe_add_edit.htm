레코드 하위 집합 보기

많은 레코드로 작업하는 경우 원하는 레코드만으로 레코드 목록을 필터링할 수 있습니다. Outlook용 CRM는 각 레코드 유형에서 선택할 수 있는 여러 가지 미리 정의된 필터를 제공합니다. 예를 들어, 모든 활성 거래처 또는 수행 중인 거래처만 볼 수 있습니다. 이러한 미리 정의된 필터를 시스템 보기라고 합니다.

목록에서 보기를 선택하면 Outlook용 CRM는 해당 보기의 탭을 만들고 목록 위에 고정합니다. 따라서 자주 사용하는 보기 사이를 쉽게 전환할 수 있습니다. 레코드를 필터링할 보기를 선택한 다음 다른 보기를 선택하여 다른 방식으로 필터링합니다.

고정 단추 를 선택하는 경우 다음에 Outlook용 CRM를 열면 보기를 사용할 수 있습니다. 이렇게 하면 대부분의 정보를 쉽게 사용할 수 있습니다.
