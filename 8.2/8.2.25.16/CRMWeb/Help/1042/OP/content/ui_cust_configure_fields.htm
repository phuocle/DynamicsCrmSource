Outlook 연락처를 Microsoft Dynamics CRM에 추가

추적함으로써 활동 레코드를 추가하는 것과 같은 방식으로 Microsoft Dynamics CRM에 연락처 레코드를 추가할 수 있습니다. 이 가이드 앞부분에서 관련 항목 설정 단추를 사용하여 영업 기회 또는 서비스 케이스 같은 특정 레코드에 활동 레코드를 연결하는 방법을 보여주었습니다. 연락처를 사용하면 같은 일을 수행할 수 있지만 대신 연락처를 Microsoft Dynamics CRM 거래처에 연결합니다.

  1. Outlook용 CRM의 탐색 창에서 사람들을 선택합니다.

  2. 최대 20개의 연락처를 선택합니다. (Shift 키를 누른 상태에서 서로 인접한 여러 연락처를 선택하거나 Ctrl 키를 누른 상태에서 서로 인접하지 않은 연락처를 선택합니다.)

  3. 리본 메뉴에서 다음 중 하나를 수행합니다.

    • 추적 단추를 선택하여 연락처를 기존 거래처 레코드에 연결하지 않고 연락처 레코드를 Microsoft Dynamics CRM에 복사합니다.

    • 상위 항목 설정 단추를 선택하여 연락처 레코드를 Microsoft Dynamics CRM에 복사하고 동시에 거래처 레코드에 연결합니다.

  4. 상위 항목 설정 단추를 선택하는 경우 레코드 조회 대화 상자에서 연락처를 연결할 거래처 이름을 선택한 다음 추가를 선택합니다.

연락처를 추적하는 경우 Outlook용 CRM는 사람의 연락처 카드 아래에 있는 추적 창을 표시합니다. 추적 창은 연락처가 추적되고 있음을 보여주고 상위 항목 설정 단추를 사용하여 레코드를 추적하는 경우 거래처 레코드에 대한 연결을 제공합니다.

팁

Outlook 양식에서 사용할 수 없는 Microsoft Dynamics CRM의 특정 필드를 채우려는 경우 CRM에서 보기 단추를 선택합니다.
