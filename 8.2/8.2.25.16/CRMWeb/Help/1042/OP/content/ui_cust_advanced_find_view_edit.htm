Outlook 양식 또는 Microsoft Dynamics CRM 양식을 열지 여부 설정

본 가이드 전체에서 추적 단추 또는 관련 항목 설정 단추(연락처에 대한 상위 항목 설정 단추)를 선택하여 전자 메일, 약속, 연락처 및 작업을 추적하는 방법을 보여줍니다. 추적은 매우 간단하며 이미 사용자에게 익숙한 동일한 Outlook 인터페이스로 작업할 수 있습니다.

또한 활동 레코드 또는 연락처 레코드를 만들고 싶을 때 Microsoft Dynamics CRM 양식을 여는 옵션이 있습니다. Microsoft Dynamics CRM 양식을 사용하는 경우 활동 및 연락처를 추적할 필요가 없습니다.

Microsoft Dynamics CRM 양식을 사용하면 누군가 Microsoft Dynamics CRM를 사용자 지정한 경우에도 유용할 수 있습니다. 예를 들어, CRM 관리에 일반 Outlook 양식을 통해 사용할 수 없는 필드가 추가되었을 수 있습니다.

새 활동의 경우 Microsoft Dynamics CRM 양식 사용

  • 개인 옵션 설정 대화 상자에서 일반 탭을 선택하고 새 활동에 사용할 양식 선택에서 하나 이상의 확인란을 선택합니다. 예를 들어 새 작업을 만들 때 Microsoft Dynamics CRM 양식을 자동으로 열고 싶은 경우 작업 확인란을 선택합니다.
