레코드의 데이터 편집

따라서 레코드의 데이터 편집을 수행하려는 경우에는 어떻게 합니까? 데이터를 편집하려면 Microsoft Dynamics CRM 양식을 열어야 합니다. 작업을 위해 Outlook용 CRM에서 나갈 필요가 없습니다. Microsoft Dynamics CRM 양식을 열려면 목록의 레코드를 두 번 클릭합니다.

Microsoft Dynamics CRM 양식이 열려 있으면 데이터를 편집하거나 일반적으로 Microsoft Dynamics CRM에서 수행하는 다른 작업을 수행할 수 있습니다.

팁

동시에 여러 레코드를 편집하려는 경우 목록에서 원하는 레코드를 선택하고 리본 메뉴에서 편집을 선택한 다음 여러 레코드 변경 창에서 원하는 필드를 편집합니다.
