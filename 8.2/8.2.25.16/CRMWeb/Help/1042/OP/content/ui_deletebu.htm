오프라인으로 전환할 때 작업을 휴대

Outlook용 CRM의 가장 큰 장점 중 하나는 분리할 때 작업을 휴대할 수 있다는 것입니다. 예를 들어, 고객 사이트로 출장을 가는 경우 인터넷에 연결하지 않고 Microsoft Dynamics CRM 연락처를 조회하고 새로운 활동을 검토하고 추가하며 보고서를 실행하고 추가 작업을 할 수 있습니다. Outlook용 CRM는 오프라인인 동안 변경한 내용을 저장하고 다시 온라인으로 전환할 때 Microsoft Dynamics CRM과 동기화합니다. 이보다 더 쉬울 수 있을까요?
