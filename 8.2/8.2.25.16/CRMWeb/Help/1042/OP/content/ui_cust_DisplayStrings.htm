읽기 창을 사용자 지정하여 섹션을 추가 또는 제거

목록에 있는 레코드를 클릭하면 레코드에 대한 세부 정보(읽기 전용)가 목록 아래 읽기 창에 표시됩니다. 읽기 창을 사용자 지정하여 섹션을 추가 또는 제거할 수 있습니다. 예를 들어, 잠재 고객 레코드의 경우 마케팅 정보 섹션을 추가하거나 제거할 수 있습니다. 읽기 창에서 정보의 순서를 변경할 수도 있습니다.

  1. 특정 레코드 종류에 대한 목록을 엽니다. 예를 들어, 잠재 고객 레코드에 대해 표시할 내용을 조정하려면 잠재 고객 목록을 엽니다.

  2. 보기 탭에서 읽기 창 사용자 지정을 선택합니다.

  3. 읽기 창 개인 설정 대화 상자에서 다음 중 하나를 수행합니다.

    • 읽기 창에서 정보를 제거하려면 표시된 섹션 아래에서 제거하려는 섹션을 선택한 다음 제거를 선택합니다.

    • 섹션을 추가하려면 사용 가능한 섹션에서 추가하려는 섹션을 선택한 다음 추가를 선택합니다.

    • 섹션의 순서를 변경하려는 경우 섹션을 선택한 다음 표시된 섹션 목록 아래의 위쪽 또는 아래쪽 화살표를 클릭합니다.
