사용자 역할에 맞는 대시보드 찾기

Outlook용 CRM에는 가장 관심을 갖고 있는 데이터와 성능 메트릭을 강조할 수 있도록 여러 가지 대시보드 레이아웃을 제공합니다. 원하는 대시보드를 찾는 가장 좋은 방법은 몇 가지를 살펴보는 것입니다.

  • 다른 대시보드 레이아웃을 보려면 대시보드 이름 옆에 있는 드롭다운 화살표를 선택하고 원하는 레이아웃을 선택합니다. 예를 들어, 대시보드에서 새로운 소식 목록을 보고 싶은 경우 영업 활동 소셜 대시보드를 시도합니다.

팁
