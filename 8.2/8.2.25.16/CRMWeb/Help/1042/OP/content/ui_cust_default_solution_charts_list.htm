받는 전자 메일을 자동으로 추적

이 가이드의 앞부분에 설명한 대로 Outlook용 CRM의 추적 이메일은 수동 프로세스입니다. 추적하려는 전자 메일을 선택한 다음 추적 단추 또는 관련 항목 설정 단추를 선택합니다. 수동 추적은 Microsoft Dynamics CRM 전자 메일과는 별도로 개인 전자 메일을 유지하는 방법을 제공합니다. 대신, 전자 메일 메시지를 자동으로 추적하는 옵션을 설정할 수 있습니다.

  1. 개인 옵션 설정 대화 상자를 열고 전자 메일 탭을 선택합니다.

  2. Outlook용 Microsoft Dynamics CRM이 Microsoft Dynamics CRM과 전자 메일을 통합하는 방식 선택 섹션에서 Outlook에서 수신 전자 메일을 확인하고 전자 메일을 Microsoft Dynamics CRM 레코드로 연결 및 저장할지 여부를 결정합니다. 옵션을 선택합니다.

  3. 이제 모든 전자 메일 메시지 또는 특정 유형만 추적할지 여부를 결정해야 합니다. 이렇게 하려면 Microsoft Dynamics CRM에서 추적할 전자 메일 메시지 선택 섹션의 추적 목록에서 옵션을 선택합니다. 대부분의 경우 CRM 전자 메일에 대한 응답으로 받은 전자 메일 메시지 옵션을 선택하게 됩니다.
