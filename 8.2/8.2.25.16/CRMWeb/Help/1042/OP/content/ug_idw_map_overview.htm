고객 센터

위치에 관계 없이 생산성 유지

사용자는 이런 사실을 이미 알고 있겠지만 Microsoft Dynamics CRM은 컴퓨터의 웹 브라우저를 사용할 뿐만 아니라 전화, Microsoft Surface 또는 iPad를 사용할 수 있습니다.

이 CRM Basics 가이드에서는 모바일 앱이 웹에서도 거의 동일하게 작동하므로 웹 브라우저의 시스템을 사용하여 가능한 짧게 유지하는 방법을 다룹니다.

전화 또는 태블릿용 온라인 저장소에서 앱을 다운로드합니다(앱을 찾으려면 “dynamics crm”을 검색). 앱을 사용하는 데 필요한 보안 권한이 있는지 일상적인 시스템 관리자에게 확인해야 할 수 있습니다.

팁

원할 경우 Microsoft Dynamics CRM은 Microsoft Outlook 내에서 작업을 수행할 수도 있습니다. 자세한 내용은 시스템 관리자에게 문의하십시오.
