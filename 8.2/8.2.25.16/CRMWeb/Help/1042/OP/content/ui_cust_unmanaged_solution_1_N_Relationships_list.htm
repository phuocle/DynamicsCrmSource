색을 사용하여 레코드 분류

색을 사용하여 Outlook의 레코드를 분류하는 방법을 알고 있는 경우 무엇을 할지 이미 알고 있는 것입니다.

  1. 범주 열에서 분류하려는 레코드 옆의 확인란을 마우스 오른쪽 단추로 클릭합니다.

  2. 색 범주를 선택합니다.

팁

색 범주의 이름을 바꾸어 보다 의미 있게 만들고 싶은 경우 모든 범주 메뉴 항목을 선택합니다. 확인란을 클릭할 때 적용할 색으로 단일 색을 설정하려면 빠른 클릭 설정 메뉴 항목을 선택합니다.
