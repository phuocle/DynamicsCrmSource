매일 대시보드로 시작

대시보드를 열어 하루를 시작할 수 있습니다. 대시보드는 사용자와 팀이 주요 메트릭(핵심 성과 지표 또는 KPI라고도 함)으로 수행하는 방법을 알 수 있는 읽기 쉬운 목록과 차트를 제공합니다.

예를 들어, 영업과 관련된 일을 하고 있고 영업 대시보드를 여는 경우 영업 파이프라인에서 열린 영업 기회의 상태 또는 마케팅 캠페인을 통해 얼마나 많은 잠재 고객이 생성되었는지 여부가 표시됩니다.

  • 대시보드를 열려면 왼쪽 탐색 창에서 해당 영역의 폴더를 엽니다. 예를 들어 영업 폴더를 엽니다. 내 작업 하위 폴더를 연 다음 대시보드 하위 폴더를 엽니다.
