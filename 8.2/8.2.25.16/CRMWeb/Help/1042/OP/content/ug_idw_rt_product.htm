Microsoft Excel 또는 다른 원본에서 데이터 가져오기

한 장소의 모든 고객 데이터를 추적하기 위해 연락처, 잠재 고객 또는 기타 레코드 종류를 전자 메일 프로그램, 스프레드시트 또는 전화 같은 다른 소스에서 Microsoft Dynamics CRM으로 가져올 수 있습니다.

참고

다른 요소에서 가져오려는 데이터의 양에 따라 데이터를 가져오는 데 약간의 계획이 필요할 수 있습니다. 시스템 관리자 가이드를 확인할 수 있습니다.

먼저, 데이터를 파일로 내보냅니다. 다음 파일 형식이 지원됩니다.

  • 쉼표로 구분된 값(.csv)

  • 텍스트(.txt)

  • 압축(.zip)

  • Microsoft Office Excel 스프레드시트 2003(.xml)

.zip 파일에 허용되는 최대 파일 크기는 32MB입니다. 다른 파일 형식의 경우 허용되는 최대 파일 크기는 8MB입니다.

데이터 가져오기 마법사를 사용하여 데이터를 가져옵니다.

  1. 설정 작업 영역에서 데이터 관리 타일을 클릭합니다. (보려면 타일 행 맨 오른쪽에 있는 오른쪽 화살표를 클릭해야 할 수 있습니다.)

  2. 가져오기를 클릭한 다음, 데이터 가져오기를 클릭합니다. 화면 지침을 따릅니다.
