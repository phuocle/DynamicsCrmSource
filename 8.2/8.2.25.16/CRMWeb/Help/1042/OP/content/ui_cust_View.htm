추가 보고서를 포함하도록 오프라인 필터 수정

이제 팀에서 소유한 모든 보고서를 선택할 수 있도록 기존 필터를 편집합니다.

  1. 현재 사용자와 같음 옆의 드롭다운 화살표를 선택한 다음 현재 사용자 또는 사용자의 팀과 같음을 선택합니다.

  2. 명령 모음에서 다른 이름으로 저장을 선택하고 필터에 모든 팀 보고서 같은 새 이름을 지정하고 설명을 추가한 다음 저장을 선택합니다.

    팁

    또한 기존 필터를 저장하고 영구히 수정할 수도 있지만 대개 새 필터를 만드는 것이 좋습니다. 이는 다른 필요를 위해 다른 필터를 사용하는 융통성이 있습니다.

  3. 새 필터 대화 상자를 닫아 오프라인으로 전환 설정 대화 상자로 돌아갑니다.

  4. 새 필터로 스크롤하는 경우 필터가 활성화된 것을 볼 수 있습니다.
