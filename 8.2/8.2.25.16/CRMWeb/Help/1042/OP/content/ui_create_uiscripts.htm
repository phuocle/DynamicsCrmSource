목록의 레코드 정렬

이것은 필요한 데이터를 찾는 최고의 방법입니다. 정렬은 레코드를 빠르게 찾을 수 있는 도구도 제공합니다.

목록의 레코드를 정렬하려면 열 머리글을 클릭하면 됩니다. 예를 들어, 아래 목록에서 거래처 이름 열 머리글을 선택하는 경우 Outlook용 CRM은 A부터 Z까지 목록을 자동으로 정렬합니다. 열 머리글을 다시 클릭하는 경우 Z부터 A까지 목록을 정렬합니다.

또한 비슷한 레코드를 함께 그룹화하도록 레코드를 정렬할 수도 있습니다. 예를 들어, 레코드 목록에 4개의 가능한 값이 있다고 가정해 보겠습니다. 레코드를 정렬할 열을 선택하는 경우 해당 필드에서 값이 같은 레코드가 모두 함께 그룹화됩니다. 이것은 레코드 하위 집합을 찾는 빠른 방법입니다.
