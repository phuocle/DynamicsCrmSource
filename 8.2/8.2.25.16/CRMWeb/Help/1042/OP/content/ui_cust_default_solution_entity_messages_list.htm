잠재 고객에게 영업 브로슈어 보내기

잠재 고객은 사용자 제품에 관심이 있습니다. 이들은 판매 브로셔를 묻는 전자 메일을 보냅니다. 판매 브로셔가 Microsoft Dynamics CRM에 저장된 경우 최소의 노력으로 Outlook용 CRM에서 직접 보낼 수 있습니다.

  1. 아직 전자 메일 메시지를 추적하지 않은 경우 추적 단추를 선택하여 추적합니다.

  2. 리본 메뉴에서 영업 홍보 자료 첨부를 선택합니다.

  3. 기타 영업 홍보 자료를 선택한 다음 거래 체결에 도움이 되는 판매 정보를 찾습니다.

참고

최근에 영업 홍보 자료를 다른 전자 메일 메시지에 첨부한 경우 목록에서 사용할 수 있습니다.
