Microsoft Dynamics CRM와(과) 레코드를 언제 동기화합니까?

Outlook용 CRM에서 레코드를 추적하면 해당 레코드의 데이터는 일반적으로 15분 내에 Microsoft Dynamics CRM에서 사용할 수 있습니다(관리자가 동기화를 설정하는 방법에 따라 즉시 사용할 수 있습니다). 레코드가 동기화된 후 사용자의 활동에 액세스하는 사람은 Microsoft Dynamics CRM에서 해당 데이터를 볼 수 있습니다.

Microsoft Dynamics CRM에 추가된 레코드도 Outlook용 CRM와 자동으로 동기화됩니다. 따라서 예를 들어 누군가 Microsoft Dynamics CRM에 활동 레코드를 추가하는 경우 Outlook용 CRM에서 이러한 레코드를 볼 수 있습니다.

기본적으로 Microsoft Dynamics CRM에서 동기화되는 레코드는 15분 간격으로 동기화됩니다. 레코드가 동기화되기를 기다리고 싶지 않으면 수동으로 동기화할 수 있습니다.

  1. Outlook용 CRM의 파일 메뉴에서 CRM을 선택합니다.

  2. 동기화 단추를 선택한 다음 동기화 시작을 선택합니다.
