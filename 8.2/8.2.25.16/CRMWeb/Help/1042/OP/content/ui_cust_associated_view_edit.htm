오프라인 필터 열기

클라이언트 사이트로 여행을 갈 준비가 되었습니다. 떠나기 전에 오프라인 필터를 수정하지 않을 경우 소유하고 있는 모든 보고서에 액세스할 수 있지만 오프라인 상태에서도 팀 보고서에 액세스할 수 있습니다. 이 예제에서는 기존 오프라인 필터를 수정한 다음 새 필터를 저장합니다.

  1. 파일 메뉴에서 CRM을 선택합니다.

  2. 오프라인으로 전환 단추를 선택한 다음 목록에서 오프라인 필터 관리를 선택합니다.

  3. 오프라인으로 전환 설정 대화 상자에서 Outlook 오프라인 보고서 필터를 선택합니다.

  4. 새 필터 창에 기존 필터를 정의하는 기준이 표시됩니다.
